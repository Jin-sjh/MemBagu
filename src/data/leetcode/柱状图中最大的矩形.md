# 柱状图中最大的矩形

## 【问题】
请解释 柱状图中最大的矩形 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/largest-rectangle-in-histogram/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772765210008-9a494ef6-d2b8-4946-ac0e-a6ec942c103b.png)

## 【回答】
**口诀速记：**

利用单调栈，为每个柱子快速找到左右两侧第一个更矮的柱子，从而计算以该柱子为高的最大矩形面积，最终取所有面积的最大值。

- 记录数组长度 n，空数组直接返回 0
- 初始化辅助数组：
  - left 数组（长度 n，初始 - 1）：存每个位置左边第一个更小高度的索引
  - right 数组（长度 n，初始 n）：存每个位置右边第一个更小高度的索引
  - 单调栈（空）：保存索引，保证对应高度递增
- 遍历数组求 left 数组：
  - 栈非空且栈顶索引高度≥当前高度，弹出栈顶（维护单调性）
  - 栈非空则 left [i]= 栈顶索引，否则 left [i]=-1，将当前索引入栈
- 清空栈，逆序遍历数组求 right 数组：
  - 栈非空且栈顶索引高度≥当前高度，弹出栈顶
  - 栈非空则 right [i]= 栈顶索引，否则 right [i]=n，将当前索引入栈
- 计算最大面积：
  - 遍历每个位置 i，宽度 = right [i]-left [i]-1，面积 = 高度 × 宽度
  - 维护最大面积，最终返回

**代码实现：**

```python
def largestRectangleArea(heights):
    n = len(heights)
    if n == 0:
        return 0  # 对应口诀1：空数组返回0
    
    left = [-1] * n   # 左边第一个比当前小的索引
    right = [n] * n   # 右边第一个比当前小的索引
    stack = []        # 单调栈，保存索引，对应的高度递增  # 对应口诀2
    
    # 求 left 数组
    for i in range(n):
        while stack and heights[stack[-1]] >= heights[i]:
            stack.pop()  # 对应口诀3：维护单调栈
        if stack:
            left[i] = stack[-1]
        else:
            left[i] = -1  # 对应口诀3：赋值left[i]
        stack.append(i)  # 对应口诀3：当前索引入栈
    
    stack = []  # 清空栈  # 对应口诀4
    
    # 求 right 数组
    for i in range(n-1, -1, -1):
        while stack and heights[stack[-1]] >= heights[i]:
            stack.pop()  # 对应口诀4：维护单调栈
        if stack:
            right[i] = stack[-1]
        else:
            right[i] = n  # 对应口诀4：赋值right[i]
        stack.append(i)  # 对应口诀4：当前索引入栈
    
    max_area = 0
    for i in range(n):
        width = right[i] - left[i] - 1  # 对应口诀5：计算宽度
        area = heights[i] * width       # 对应口诀5：计算面积
        if area > max_area:
            max_area = area  # 对应口诀5：维护最大面积
    return max_area  # 对应口诀5：返回结果
```

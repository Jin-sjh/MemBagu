# 从前序与中序遍历序列构造二叉树

## 【问题】
请解释 从前序与中序遍历序列构造二叉树 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772770835839-3d6e38d0-f0b9-4e3b-852c-d43c0f25d2f5.png)

## 【回答】
**口诀速记：**

利用前序遍历首个元素为根节点的特性确定当前子树根节点，通过哈希表快速定位该根节点在中序遍历中的位置，从而划分出左右子树的元素区间，再递归地为左右子树重复此过程，最终构建完整二叉树。

- 构建中序遍历的哈希映射表（值→索引），实现 O (1) 查找根节点位置，记录前序 / 中序数组长度 n
- 定义递归构建函数helper（参数：前序左边界 pre\_left、前序右边界 pre\_right、中序左边界 in\_left、中序右边界 in\_right）：
- 终止条件：pre\_left > pre\_right（区间为空，返回 None）
- 取前序左边界值作为当前根节点值，创建根节点
- 从哈希表中找到根节点在中序的位置 in\_pos，计算左子树长度 left\_size = in\_pos - in\_left
- 递归构建左子树：前序区间 [pre\_left+1, pre\_left+left\_size]、中序区间 [in\_left, in\_pos-1]
- 递归构建右子树：前序区间 [pre\_left+left\_size+1, pre\_right]、中序区间 [in\_pos+1, in\_right]
- 返回当前根节点
- 调用 helper 函数，传入初始边界（0, n-1, 0, n-1），返回构建好的二叉树根节点

**代码实现：**

```python
# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def buildTree(self, preorder: list[int], inorder: list[int]) -> TreeNode | None:
        # 用哈希表存储中序遍历中值到索引的映射，O(1)查找
        inorder_map = {val: idx for idx, val in enumerate(inorder)}  # 对应口诀1：构建哈希映射
        n = len(preorder)  # 对应口诀1：记录数组长度
        
        def helper(pre_left, pre_right, in_left, in_right):
            # 递归终止条件：区间为空
            if pre_left > pre_right:
                return None  # 对应口诀2：终止条件
            # 前序遍历的第一个节点是当前子树的根
            root_val = preorder[pre_left]
            root = TreeNode(root_val)  # 对应口诀2：创建根节点
            # 在中序遍历中找到根节点的位置
            in_pos = inorder_map[root_val]  # 对应口诀2：查找根节点中序位置
            # 左子树的长度
            left_size = in_pos - in_left  # 对应口诀2：计算左子树长度
            # 递归构造左子树
            root.left = helper(pre_left + 1, pre_left + left_size, in_left, in_pos - 1)  # 对应口诀2：构建左子树
            # 递归构造右子树
            root.right = helper(pre_left + left_size + 1, pre_right, in_pos + 1, in_right)  # 对应口诀2：构建右子树
            return root  # 对应口诀2：返回当前根节点
        
        return helper(0, n - 1, 0, n - 1)  # 对应口诀3：调用递归函数
```

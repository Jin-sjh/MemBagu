# 合并两个有序链表

## 【问题】
请解释 合并两个有序链表 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/merge-two-sorted-lists/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772711503965-cd4c987d-cd8a-4702-8d28-b4d140f368c4.png)

## 【回答】
**口诀速记：**

通过\*\*哨兵节点简化头节点处理，配合双指针遍历两个有序链表\*\*，每次选择当前值更小的节点接入新链表，最后直接拼接未遍历完的链表剩余部分，在 O(1) 额外空间下完成 O(n+m) 时间复杂度的有序合并。

- 建哨兵节点（哑节点），初始化游标指针指向它
- 双指针循环遍历两链表（均非空时）
- 取值较小的节点接入游标指针后，移动对应链表指针
- 移动游标指针到新接入的节点
- 循环结束后，拼接剩余未遍历的链表节点
- 返回哨兵节点的下一个节点（新链表头）

**解法1：**

```python
# Definition for singly-linked list.
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def mergeTwoLists(self, list1: ListNode, list2: ListNode) -> ListNode:
        # 哨兵节点，简化头节点处理
        dummy = ListNode(-1)
        # 游标指针，用于构建新链表
        curr = dummy  # 对应口诀1：建哨兵节点+初始化游标指针
        
        # 双指针循环，直到其中一个链表遍历完毕
        while list1 and list2:  # 对应口诀2：双指针循环遍历
            if list1.val <= list2.val:
                curr.next = list1
                list1 = list1.next  # 移动list1指针
            else:
                curr.next = list2
                list2 = list2.next  # 移动list2指针  # 对应口诀3：接入较小节点+移动对应指针
            curr = curr.next  # 移动游标指针  # 对应口诀4：移动游标指针
        
        # 拼接剩余节点（无需循环，直接接入即可）
        curr.next = list1 if list1 else list2  # 对应口诀5：拼接剩余节点
        
        # 返回新链表的真实头节点
        return dummy.next  # 对应口诀6：返回新链表头
```

**解法2：**

```python
class Solution:
    def mergeTwoLists(self, list1: ListNode, list2: ListNode) -> ListNode:
        # 递归终止条件：其中一个链表为空，返回另一个
        if not list1:
            return list2
        if not list2:
            return list1
        
        # 递归选择较小值的节点，拼接后续结果
        if list1.val <= list2.val:
            list1.next = self.mergeTwoLists(list1.next, list2)
            return list1
        else:
            list2.next = self.mergeTwoLists(list1, list2.next)
            return list2
```

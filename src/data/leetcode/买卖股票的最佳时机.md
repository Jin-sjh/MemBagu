# 买卖股票的最佳时机

## 【问题】
请解释 买卖股票的最佳时机 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772771365708-5209f340-dcf3-40e0-9de7-73c967a0dac3.png)

## 【回答】
**口诀速记：**

在遍历股价数组的过程中，始终记录当前遇到的最低股价作为最优买入点，同时计算当前股价与该最低点的差值（即潜在利润），并持续更新最大利润值，最终得到一次买卖能获得的最大收益。  【一低一高一次遍历，先更利润再更价格】

- 若价格数组长度小于 2，直接返回 0（无法完成买卖操作，利润为 0）
- 初始化两个关键变量：
- min_price 为数组第一个价格（记录遍历过程中遇到的最低买入价）
- max_profit 为 0（记录最大可获得利润，初始无利润）
- 遍历数组中从第二个开始的每个价格：
- 计算当前价格卖出的利润（当前价格 - 最低买入价）
- 若当前利润大于max_profit，更新max_profit为当前利润
- 若当前价格低于min_price，更新min_price为当前价格（更新最低买入价）
- 遍历结束后返回max_profit（即一次买卖能获得的最大利润）

**代码实现：**

```python
def maxProfit(prices):
    if len(prices) < 2:
        return 0  # 对应口诀1：长度不足返回0
    
    min_price = prices[0]  # 对应口诀2：初始化最低价格
    max_profit = 0         # 对应口诀2：初始化最大利润
    
    for price in prices[1:]:  # 对应口诀3：遍历第二个及以后的价格
        # 计算当前卖出的利润
        current_profit = price - min_price  # 对应口诀3：计算当前利润
        # 更新最大利润
        if current_profit > max_profit:
            max_profit = current_profit  # 对应口诀3：更新最大利润
        # 更新最低价格
        if price < min_price:
            min_price = price  # 对应口诀3：更新最低价格
    
    return max_profit  # 对应口诀4：返回最大利润
```

# 乘积最大子数组

## 【问题】
请解释 乘积最大子数组 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/maximum-product-subarray/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772774809017-e0819452-fc06-4a5f-847a-073ac74cf94c.png)

## 【回答】
**口诀速记：**

利用动态规划，同时维护以当前元素结尾的子数组的最大和最小乘积，以应对负数相乘可能产生更大正数的特殊情况，从而在单次遍历中找到全局最大乘积。【初始定最值，遍历更全局】

**代码实现：**

```python
def maxProduct(nums):
    if not nums:
        return 0
    
    # 初始化：以第一个元素结尾的最大、最小乘积就是它本身
    max_so_far = curr_max = curr_min = nums[0]
    
    for num in nums[1:]:
        # 用临时变量保存，避免 curr_max 被覆盖后影响 curr_min 的计算
        temp_max = curr_max
        temp_min = curr_min
        
        # 计算以当前 num 结尾的最大和最小乘积
        curr_max = max(num, temp_max * num, temp_min * num)
        curr_min = min(num, temp_max * num, temp_min * num)
        
        # 更新全局最大乘积
        max_so_far = max(max_so_far, curr_max)
    
    return max_so_far
```

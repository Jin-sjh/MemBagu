# 排序链表

## 【问题】
请解释 排序链表 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/sort-list/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772773932452-2d0cda3b-2fcc-4518-9a9b-84feeb0ee12a.png)

## 【回答】
**口诀速记：**

利用归并排序 “分而治之” 的思路，通过快慢指针将链表递归拆分至最小单元，再有序合并，从而在 O (n log n) 时间复杂度内完成链表排序。

【判终止，找中点，切两半 递归排，合并接，返结果】

**代码实现：**

```python
# Definition for singly-linked list.
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def sortList(self, head: ListNode) -> ListNode:
        # 递归终止条件：链表为空或只有一个节点
        if not head or not head.next:
            return head
        
        # 1. 使用快慢指针找到中点，分割链表
        slow, fast = head, head.next
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
        # 中点是 slow.next，断开链表
        mid = slow.next
        slow.next = None  # 切断链表
        
        # 2. 递归排序左右两部分
        left = self.sortList(head)
        right = self.sortList(mid)
        
        # 3. 合并两个有序链表
        dummy = ListNode(0)
        curr = dummy
        while left and right:
            if left.val < right.val:
                curr.next = left
                left = left.next
            else:
                curr.next = right
                right = right.next
            curr = curr.next
        # 合并剩余部分
        curr.next = left if left else right
        
        return dummy.next
```

# 最长递增子序列

## 【问题】
请解释 最长递增子序列 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/longest-increasing-subsequence/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772777999950-521991d6-8f5e-4cf8-834d-864955719acf.png)

## 【回答】
**口诀速记：**

- DP 数组初始全为 1
- 双层遍历比大小
- 后大前则更 DP 值
- 最终返回 DP 最大值

**代码实现：**

```python
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        n = len(nums)

        dp = [1] * n  # 对应口诀1：DP数组初始全为1

        for i in range(1, n):  # 对应口诀2：双层遍历比大小（外层遍历）
            for j in range(i):  # 对应口诀2：双层遍历比大小（内层遍历）
                if nums[i] > nums[j]:  # 对应口诀3：后大前则更DP值（比大小）
                    dp[i] = max(dp[i], dp[j]+1)  # 对应口诀3：后大前则更DP值（更DP值）
        return max(dp)  # 对应口诀4：最终返回DP最大值
```

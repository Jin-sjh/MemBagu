# 电话号码的字母组合

## 【问题】
请解释 电话号码的字母组合 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/letter-combinations-of-a-phone-number/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772710674228-003492a5-95f4-43a0-bbfd-bbf200ae7d1b.png)

## 【回答】
**口诀速记：**

通过回溯（或迭代）遍历所有数字对应的字母选择，逐步构建并生成所有可能的字母组合。

- 空数字串直接返回空列表
- 定义数字到字母的映射字典
- 初始化结果列表，定义回溯函数（参数：索引、路径）
- 回溯终止：索引到数字串末尾，路径转字符串加入结果
- 取当前数字对应的字母，遍历每个字母
- 选择：字母加入路径，递归处理下一个数字
- 回溯：移除路径最后一个字母（撤销选择）
- 启动回溯（索引 0、空路径），返回结果列表

**代码实现：**

```python
def letterCombinations(digits: str) -> list[str]:
    if not digits:
        return []  # 对应口诀1：空数字串返回空列表
    
    # 数字到字母的映射
    phone_map = {
        '2': 'abc',
        '3': 'def',
        '4': 'ghi',
        '5': 'jkl',
        '6': 'mno',
        '7': 'pqrs',
        '8': 'tuv',
        '9': 'wxyz'
    }  # 对应口诀2：定义数字字母映射
    
    result = []  # 初始化结果列表
    
    def backtrack(index, path):  # 对应口诀3：定义回溯函数
        # 终止条件：索引到达末尾，说明组合完成
        if index == len(digits):
            result.append(''.join(path))
            return  # 对应口诀4：回溯终止逻辑
        
        # 获取当前数字对应的所有字母
        current_digit = digits[index]
        for char in phone_map[current_digit]:  # 对应口诀5：遍历当前数字的字母
            # 选择：将当前字母加入路径
            path.append(char)
            # 递归：处理下一个数字
            backtrack(index + 1, path)  # 对应口诀6：选择+递归
            # 回溯：撤销选择，移除最后加入的字母
            path.pop()  # 对应口诀7：回溯撤销选择
    
    # 从索引0开始，初始路径为空
    backtrack(0, [])  # 对应口诀8：启动回溯
    return result  # 对应口诀8：返回结果列表
```

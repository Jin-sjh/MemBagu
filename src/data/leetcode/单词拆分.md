# 单词拆分

## 【问题】
请解释 单词拆分 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/word-break/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772773310110-d9077558-89b0-4b3a-8358-8527fed62441.png)

## 【回答】
**口诀速记：**

用动态规划记录前 i 个字符能否被字典单词拼接，通过检查所有可能的前序合法位置 j，判断从 j 到 i 的子串是否在字典中，从而递推出整个字符串的可拆分性。

- 将单词字典转为集合word_set（实现 O (1) 时间查找），记录字符串长度n
- 初始化 DP 数组（长度 n+1，全为 False），设置边界dp[0] = True（空字符串可被拼接，作为基准）
- 遍历字符串结束位置i（从 1 到 n）：
- 遍历分割点j（从 0 到 i-1）：✅ 若dp[j]为 True（前 j 个字符可被拼接）且子串s[j:i]在word_set中：✅ 标记dp[i] = True，并跳出内层循环（找到可行解即可，无需继续）
- 遍历结束后返回dp[n]（表示整个字符串是否可被字典单词拼接）

**代码实现：**

```python
def wordBreak(s: str, wordDict: list[str]) -> bool:
    word_set = set(wordDict)  # 对应口诀1：字典转集合（快速查找）
    n = len(s)                # 对应口诀1：记录字符串长度
    dp = [False] * (n + 1)    # 对应口诀2：初始化DP数组
    dp[0] = True  # 空字符串可以被拼接  # 对应口诀2：设置边界基准

    for i in range(1, n + 1):  # 对应口诀3：遍历结束位置i
        for j in range(i):     # 对应口诀3：遍历分割点j
            if dp[j] and s[j:i] in word_set:  # 对应口诀3：判定拼接可行性
                dp[i] = True
                break  # 找到一个可行解即可，无需继续检查  # 对应口诀3：跳出内层循环
    return dp[n]  # 对应口诀4：返回整个字符串的拼接结果
```

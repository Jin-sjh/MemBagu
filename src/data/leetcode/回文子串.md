# 回文子串

## 【问题】
请解释 回文子串 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/palindromic-substrings/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772782687295-d7a298cd-a670-41b5-9329-ff1a696daade.png)

## 【回答】
**口诀速记：**

以每个可能的中心（单字符或双字符）向左右扩展，统计所有满足回文条件的子串数量。

- 初始化计数为 0
- 定义扩展函数判回文
- 边界内字符相等则计数 + 1
- 向左右扩展继续判
- 遍历每个字符当中心
- 先扩奇数长度回文
- 再扩偶数长度回文
- 最终返回总计数

**代码实现：**

```python
def countSubstrings(s: str) -> int:
    n = len(s)
    count = 0  # 对应口诀1：初始化计数为0
    
    def expand(left: int, right: int) -> None:
        nonlocal count
        # 对应口诀2：定义扩展函数判回文
        while left >= 0 and right < n and s[left] == s[right]:
            count += 1  # 对应口诀3：边界内字符相等则计数+1
            left -= 1
            right += 1  # 对应口诀4：向左右扩展继续判
    
    for i in range(n):  # 对应口诀5：遍历每个字符当中心
        # 奇数长度回文，中心为 i
        expand(i, i)  # 对应口诀6：先扩奇数长度回文
        # 偶数长度回文，中心为 i 和 i+1
        expand(i, i + 1)  # 对应口诀7：再扩偶数长度回文
    
    return count  # 对应口诀8：最终返回总计数
```

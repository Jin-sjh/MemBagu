# 验证二叉搜索树

## 【问题】
请解释 验证二叉搜索树 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/validate-binary-search-tree/

## 【回答】
**口诀速记：**

利用二叉搜索树中序遍历结果严格递增的性质，或通过递归验证每个节点值在其允许的上下边界内，从而判断树的有效性。

- 定义二叉树节点类，解法类中初始化前驱值prev为负无穷（记录中序遍历的前一个节点值）
- 定义中序遍历递归函数inorder（参数：当前节点 node）：
- 终止条件：节点为空，返回 True（空树是合法 BST）
- 递归遍历左子树，若左子树返回 False，直接返回 False（左子树不合法则整体不合法）
- 检查当前节点值：若≤前驱值，返回 False（BST 中序遍历需严格递增）
- 更新前驱值为当前节点值，递归遍历右子树并返回结果
- 调用中序遍历函数，传入根节点 root，返回最终结果（是否为合法 BST）

**解法1：**

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def isValidBST(self, root: TreeNode) -> bool:
        self.prev = float('-inf')  # 对应口诀1：初始化前驱为负无穷
        
        def inorder(node):
            if not node:
                return True  # 对应口诀2：终止条件（空节点返回True）
            # 遍历左子树
            if not inorder(node.left):
                return False  # 对应口诀2：左子树不合法则返回False
            # 检查当前节点是否大于前驱
            if node.val <= self.prev:
                return False  # 对应口诀2：值不递增则返回False
            self.prev = node.val  # 对应口诀2：更新前驱值
            # 遍历右子树
            return inorder(node.right)  # 对应口诀2：返回右子树遍历结果
        
        return inorder(root)  # 对应口诀3：调用函数并返回结果
```

**解法2：**

```python
class Solution:
    def isValidBST(self, root: TreeNode) -> bool:
        stack = []
        prev = float('-inf')
        
        while stack or root:
            # 一直向左走到最左节点
            while root:
                stack.append(root)
                root = root.left
            # 弹出并处理节点
            root = stack.pop()
            if root.val <= prev:
                return False
            prev = root.val
            # 转向右子树
            root = root.right
        
        return True
```

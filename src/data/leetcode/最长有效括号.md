# 最长有效括号

## 【问题】
请解释 最长有效括号 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/longest-valid-parentheses/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772712378833-73d81c97-667c-4692-85fe-cf69e574de70.png)

## 【回答】
**口诀速记：**

通过\*\*跟踪括号的匹配关系\*\*（栈记录未匹配括号索引 / 双指针计数左右括号），在遍历过程中动态计算以当前位置结尾的有效括号长度，最终找到全局最大值。

- 初始化栈（存入基准索引 - 1）、最大长度为 0
- 遍历字符串，记录索引和字符
- 遇左括号：将当前索引压入栈
- 遇右括号：弹出栈顶元素
- 栈空则压入当前索引（更新基准）
- 栈非空则计算当前有效长度（i - 栈顶），更新最大长度
- 遍历结束返回最大长度

**解法1：**

```python
def longestValidParentheses(s: str) -> int:
    stack = [-1]  # 初始化栈，放入基准索引-1
    max_len = 0  # 对应口诀1：初始化栈和最大长度
    for i, char in enumerate(s):  # 对应口诀2：遍历字符串（索引+字符）
        if char == '(':
            stack.append(i)  # 对应口诀3：左括号压入索引
        else:
            stack.pop()  # 对应口诀4：右括号弹出栈顶
            if not stack:
                stack.append(i)  # 对应口诀5：栈空则压入当前索引（更新基准）
            else:
                max_len = max(max_len, i - stack[-1])  # 对应口诀6：计算有效长度并更新最大值
    return max_len  # 对应口诀7：返回最大长度
```

**解法2：**

```python
def longestValidParentheses(s: str) -> int:
    max_len = 0
    left = right = 0
    
    # 从左到右遍历
    for char in s:
        if char == '(':
            left += 1
        else:
            right += 1
        if left == right:
            max_len = max(max_len, 2 * right)
        elif right > left:
            left = right = 0
    
    left = right = 0
    # 从右到左遍历
    for char in reversed(s):
        if char == '(':
            left += 1
        else:
            right += 1
        if left == right:
            max_len = max(max_len, 2 * left)
        elif left > right:
            left = right = 0
    
    return max_len
```

# 对称二叉树

## 【问题】
请解释 对称二叉树 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/symmetric-tree/

## 【回答】
**口诀速记：**

通过递归或迭代的方式，检查二叉树根节点的左子树与右子树是否互为镜像，即左子树的左节点对应右子树的右节点，左子树的右节点对应右子树的左节点，且对应节点值相等。

- 定义二叉树节点类，解法类中实现主函数isSymmetric：
- 若根节点为空，直接返回 True（空树是对称的）
- 调用镜像校验函数isMirror，传入根节点的左、右子树，返回校验结果
- 定义镜像校验函数isMirror（参数：左节点 left、右节点 right）：
- 终止条件 1：两个节点都为空，返回 True（对称）
- 终止条件 2：仅一个节点为空，返回 False（不对称）
- 核心校验：节点值相等，且左节点的左子树与右节点的右子树镜像、左节点的右子树与右节点的左子树镜像，三者同时满足则返回 True
- 最终返回镜像校验的结果（是否为对称二叉树）

**解法1：**

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def isSymmetric(self, root: TreeNode) -> bool:
        if not root:
            return True  # 对应口诀1：空树对称
        return self.isMirror(root.left, root.right)  # 对应口诀1：调用镜像校验
    
    def isMirror(self, left: TreeNode, right: TreeNode) -> bool:
        # 两个都为空
        if not left and not right:
            return True  # 对应口诀2：双空节点对称
        # 一个为空，一个不为空
        if not left or not right:
            return False  # 对应口诀2：单空节点不对称
        # 值相等，且子树镜像
        return (left.val == right.val and
                self.isMirror(left.left, right.right) and
                self.isMirror(left.right, right.left))  # 对应口诀2：核心镜像校验
```

**解法2：**

```python
from collections import deque

class Solution:
    def isSymmetric(self, root: TreeNode) -> bool:
        if not root:
            return True
        
        q = deque()
        q.append(root.left)
        q.append(root.right)
        
        while q:
            left = q.popleft()
            right = q.popleft()
            
            if not left and not right:
                continue
            if not left or not right:
                return False
            if left.val != right.val:
                return False
            
            q.append(left.left)
            q.append(right.right)
            q.append(left.right)
            q.append(right.left)
        
        return True
```

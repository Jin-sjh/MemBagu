# 二叉树中的最大路径和

## 【问题】
请解释 二叉树中的最大路径和 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/binary-tree-maximum-path-sum/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772771984207-56c0679b-457d-4b3c-8383-fae040d14730.png)

## 【回答】
**口诀速记：**

通过\*\*后序遍历递归计算每个节点向父节点贡献的最大路径和（舍弃负贡献）\*\*，同时在遍历过程中更新以当前节点为最高点的路径和，最终得到全局最大路径和。

- 初始化类属性max_sum为负无穷（兼容全负数节点的情况，确保能取到最大负值）
- 定义 DFS 递归函数（参数：当前节点 node），核心是计算 “节点贡献值” 和 “节点为顶的路径和”：
- 终止条件：节点为空，返回贡献值 0（空节点无值，贡献为 0）
- 递归计算左右子树的贡献值：用max(递归结果, 0)舍弃负数贡献（负数贡献会拉低路径和，不如不取）
- 计算以当前节点为最高点的路径和：节点值 + 左贡献 + 右贡献
- 更新全局最大路径和max_sum（取当前max_sum和 “节点为顶的路径和” 的较大值）
- 返回当前节点能向上层贡献的最大路径和：节点值 + 左/右贡献的最大值（只能选一条路径向上）
- 调用 DFS 函数传入根节点 root，最终返回max_sum（整棵树的最大路径和）

**代码实现：**

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def maxPathSum(self, root: TreeNode) -> int:
        # 初始化全局最大路径和为负无穷，处理全是负数的情况
        self.max_sum = float('-inf')  # 对应口诀1：初始化全局最大值
        
        def dfs(node):
            if not node:
                return 0  # 对应口诀2：终止条件（空节点贡献0）
            
            # 递归计算左右子树的贡献值，舍弃负数
            left_gain = max(dfs(node.left), 0)  # 对应口诀2：左子树贡献（舍弃负数）
            right_gain = max(dfs(node.right), 0)  # 对应口诀2：右子树贡献（舍弃负数）
            
            # 以当前节点为最高点的路径和
            current_path_sum = node.val + left_gain + right_gain  # 对应口诀2：计算节点为顶的路径和
            # 更新全局最大路径和
            self.max_sum = max(self.max_sum, current_path_sum)  # 对应口诀2：更新全局最大值
            
            # 返回当前节点能向上贡献的最大路径和
            return node.val + max(left_gain, right_gain)  # 对应口诀2：返回向上贡献值
        
        dfs(root)  # 对应口诀3：调用DFS遍历整棵树
        return self.max_sum  # 对应口诀3：返回全局最大路径和
```

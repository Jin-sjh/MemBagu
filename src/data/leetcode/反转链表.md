# 反转链表

## 【问题】
请解释 反转链表 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/reverse-linked-list/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772775961328-226ee490-f517-4ecf-9512-b245ad58c3fb.png)

## 【回答】
**口诀速记：**

通过迭代或递归，逐个改变链表节点的 next 指针方向，使原本指向后继的指针转而指向前驱，从而将整个链表的顺序颠倒。

【

- 前空当前指表头
- 循环存下反转指
- 双指针依次后移
- 循环完返前节点
- 】

**解法1：**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        prev = None  # 对应口诀1：前空当前指表头（前指针初始为空）
        curr = head  # 对应口诀1：前空当前指表头（当前指针指向头节点）
        
        while curr:  # 对应口诀2：循环存下反转指（循环起始）
            next_temp = curr.next  # 对应口诀2：循环存下反转指（存下一个节点）
            curr.next = prev       # 对应口诀2：循环存下反转指（反转指针）
            prev = curr            # 对应口诀3：双指针依次后移（前指针后移）
            curr = next_temp       # 对应口诀3：双指针依次后移（当前指针后移）
        
        return prev  # 对应口诀4：循环完返前节点
```

**解法2：**

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        # 递归终止条件：当前节点为空或下一个节点为空
        if not head or not head.next:
            return head
        
        # 递归反转后续链表，返回新的头节点
        new_head = self.reverseList(head.next)
        
        # 反转当前节点和下一个节点的指针
        head.next.next = head
        head.next = None
        
        return new_head
```

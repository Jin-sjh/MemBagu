# 只出现一次的数字

## 【问题】
请解释 只出现一次的数字 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/single-number/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772773201652-bbce2fd1-f621-4f33-8fb4-455575c9fdab.png)

## 【回答】
**口诀速记：**

利用\*\*异或运算\*\*的自反性（a ^ a = 0）和恒等性（a ^ 0 = a），将数组中所有元素依次异或，成对出现的数字会相互抵消为 0，最终结果即为只出现一次的数字。  【任何数：1.和0异或是本身； 2.和自身异或是0； 3.满足交换律】

- 初始化结果变量res为 0（异或运算的基准值，0 异或任何数等于该数本身）
- 遍历数组中的每一个数字num：
- 对res和当前数字执行异或操作（res ^= num）
- 异或特性：相同数字异或结果为 0，0 异或目标数字结果为目标数字
- 遍历结束后返回res（即为数组中唯一出现一次的数字）

**代码实现：**

```python
def singleNumber(nums):
    res = 0  # 对应口诀1：初始化异或基准值
    for num in nums:  # 对应口诀2：遍历数组所有数字
        res ^= num    # 对应口诀2：执行异或操作
    return res        # 对应口诀3：返回唯一出现的数字
```

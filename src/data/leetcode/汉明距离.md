# 汉明距离

## 【问题】
请解释 汉明距离 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/hamming-distance/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772780630050-f0bdbaa6-2ae9-4f0a-a090-d0b373999eaa.png)

## 【回答】
**口诀速记：**

通过异或运算定位两数二进制不同位，再用位运算技巧统计异或结果中 1 的个数，即为汉明距离。

- 两数异或找不同位
- 初始化计数为 0
- 循环消最低位 1
- 消一次计数加 1
- 最终返回计数值

**代码实现：**

```python
def hammingDistance(x: int, y: int) -> int:
    # 计算异或
    xor = x ^ y  # 对应口诀1：两数异或找不同位
    # 统计异或结果中1的个数
    count = 0  # 对应口诀2：初始化计数为0
    while xor:
        # 消除最低位的1
        xor &= xor - 1  # 对应口诀3：循环消最低位1
        count += 1  # 对应口诀4：消一次计数加1
    return count  # 对应口诀5：最终返回计数值
```

# 目标和

## 【问题】
请解释 目标和 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/target-sum/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772780767371-11fc007d-3796-4b03-8904-d7c48c2ec503.png)

## 【回答】
**口诀速记：**

通过数学转化，将 “给每个数加正负号凑目标和” 的问题，转化为 “在数组中选取若干数，使其和等于 (总和 + 目标值) / 2” 的 0-1 背包问题，再用动态规划求解

- 算总和判无解情况
- 求 P 值转背包问题
- DP 数组初始 0，零位设 1
- 逆序遍历更 DP 值
- 最终返回 DP [P] 值

**代码实现：**

```python
def findTargetSumWays(nums, target):
    total_sum = sum(nums)
    # 无解的情况
    if (total_sum + target) % 2 != 0 or (total_sum + target) < 0:
        return 0  # 对应口诀1：算总和判无解情况
    P = (total_sum + target) // 2  # 对应口诀2：求P值转背包问题
    dp = [0] * (P + 1)
    dp[0] = 1  # 初始条件  # 对应口诀3：DP数组初始0，零位设1
    for num in nums:
        # 从后往前遍历，避免重复使用同一个元素
        for i in range(P, num - 1, -1):  # 对应口诀4：逆序遍历更DP值（逆序遍历）
            dp[i] += dp[i - num]  # 对应口诀4：逆序遍历更DP值（更DP值）
    return dp[P]  # 对应口诀5：最终返回DP[P]值
```

# 括号生成

## 【问题】
请解释 括号生成 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/generate-parentheses/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772711685216-6e8a4daa-91d2-4423-bac2-75a4fbe3a9bf.png)

## 【回答】
**口诀速记：**

通过回溯法在构造括号字符串的过程中，始终遵循「左括号数量不超过 n、右括号数量不超过左括号数量」的约束，穷举所有有效组合。

- 初始化结果列表，定义回溯函数（参数：当前串、左括号数、右括号数）
- 回溯终止：当前串长度为 2n，加入结果列表
- 左括号数 < n 时，添加左括号并递归（左数 + 1）
- 右括号数 < 左括号数时，添加右括号并递归（右数 + 1）
- 启动回溯（空串、0 左、0 右），返回结果列表

**代码实现：**

```python
def generateParenthesis(n: int) -> list[str]:
    result = []  # 初始化结果列表
    
    def backtrack(current: str, open: int, close: int):
        # 终止条件：字符串长度达到 2n
        if len(current) == 2 * n:
            result.append(current)
            return  # 对应口诀2：回溯终止逻辑
        
        # 尝试添加左括号
        if open < n:
            backtrack(current + '(', open + 1, close)  # 对应口诀3：添加左括号并递归
        
        # 尝试添加右括号
        if close < open:
            backtrack(current + ')', open, close + 1)  # 对应口诀4：添加右括号并递归
    
    backtrack("", 0, 0)  # 对应口诀5：启动回溯
    return result  # 对应口诀5：返回结果列表

# 测试
print(generateParenthesis(3))  # 输出: ["((()))","(()())","(())()","()(())","()()()"]
print(generateParenthesis(1))  # 输出: ["()"]
```

# 找到所有数组中消失的数字

## 【问题】
请解释 找到所有数组中消失的数字 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/find-all-numbers-disappeared-in-an-array/

## 【回答】
**口诀速记：**

利用数组自身的索引作为 “哈希表”，通过将对应位置的元素标记为负数来记录数字的出现，最后通过正数位置反推缺失的数字。

- 遍历数组取绝对值
- 对应位置标为负数
- 收集正数索引加 1
- 即为缺失的数字

**代码实现：**

```python
def findDisappearedNumbers(nums):
    n = len(nums)
    for i in range(n):
        # 取绝对值，因为之前可能已经被标记为负数
        num = abs(nums[i])  # 对应口诀1：遍历数组取绝对值
        # 将对应位置的数标记为负数
        if nums[num - 1] > 0:
            nums[num - 1] = -nums[num - 1]  # 对应口诀2：对应位置标为负数
    
    # 收集所有正数对应的索引 +1，即为缺失的数字
    return [i + 1 for i in range(n) if nums[i] > 0]  # 对应口诀3：收集正数索引加1 + 口诀4：即为缺失的数字
```

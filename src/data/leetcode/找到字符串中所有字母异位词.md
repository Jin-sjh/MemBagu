# 找到字符串中所有字母异位词

## 【问题】
请解释 找到字符串中所有字母异位词 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/find-all-anagrams-in-a-string/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772780309357-9f99cad8-9eb7-4895-a20f-7d8148508787.png)

## 【回答】
**口诀速记：**

利用滑动窗口维护与目标串长度相同的字符计数窗口，通过比较字符计数是否一致来快速定位所有异位词的起始位置。

- p 更长直接返空结果
- 初始化两字符计数数组
- 统计 p 的字符计数
- 滑动窗口左右边界始为 0
- 右移加字符到计数
- 窗超长度左移减字符
- 窗等长度判计数相等
- 相等则存左边界到结果

**解法1：**

```python
def findAnagrams(s: str, p: str) -> list[int]:
    res = []
    if len(p) > len(s):
        return res  # 对应口诀1：p更长直接返空结果
    
    # 初始化两个计数数组
    count_p = [0] * 26
    count_s = [0] * 26  # 对应口诀2：初始化两字符计数数组
    
    # 统计 p 的字符计数
    for c in p:
        count_p[ord(c) - ord('a')] += 1  # 对应口诀3：统计p的字符计数
    
    # 初始化滑动窗口
    left = 0  # 对应口诀4：滑动窗口左右边界始为0（左边界初始化）
    for right in range(len(s)):  # 对应口诀4：滑动窗口左右边界始为0（右边界遍历）
        # 加入右边字符
        count_s[ord(s[right]) - ord('a')] += 1  # 对应口诀5：右移加字符到计数
        
        # 当窗口大小超过 p 的长度时，移动左边界
        if right - left + 1 > len(p):
            count_s[ord(s[left]) - ord('a')] -= 1
            left += 1  # 对应口诀6：窗超长度左移减字符
        
        # 检查当前窗口是否是异位词
        if right - left + 1 == len(p) and count_s == count_p:
            res.append(left)  # 对应口诀7：窗等长度判计数相等 + 口诀8：相等则存左边界到结果
    
    return res
```

**解法2：**

```python
def findAnagrams(s: str, p: str) -> list[int]:
    res = []
    if len(p) > len(s):
        return res
    
    count = [0] * 26
    # 统计 p 的字符计数
    for c in p:
        count[ord(c) - ord('a')] += 1
    
    left = 0
    match = 0  # 记录当前窗口与 p 字符计数相同的字母个数
    
    for right in range(len(s)):
        # 处理右边进入窗口的字符
        r_idx = ord(s[right]) - ord('a')
        count[r_idx] -= 1
        if count[r_idx] == 0:
            match += 1
        
        # 当窗口大小超过 p 的长度时，处理左边移出窗口的字符
        if right - left + 1 > len(p):
            l_idx = ord(s[left]) - ord('a')
            if count[l_idx] == 0:
                match -= 1
            count[l_idx] += 1
            left += 1
        
        # 如果 26 个字母的计数都匹配，说明找到异位词
        if match == 26:
            res.append(left)
    
    return res
```

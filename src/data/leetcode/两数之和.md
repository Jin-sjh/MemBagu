# 两数之和

## 【问题】
请解释 两数之和 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/two-sum/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772846308135-3110cb3f-bcec-498d-b5a5-337d4c670c03.png)

## 【回答】
**口诀速记：**

利用哈希表存储已遍历元素及其索引，在单次遍历中通过查找 “补数”（目标值与当前元素的差）是否存在，实现 O (n) 时间复杂度的两数之和求解。

- 初始化字典存数字和索引
- 遍历数组取索引与数值
- 计算当前数的补数
- 补数在字典则返回双索引
- 补数不在则存当前数和索引
- 最终返回结果列表

**代码实现：**

```python
def twoSum(nums, target):
    # 创建一个空字典，用于存储 {数字: 索引}
    num_map = {}  # 对应口诀1：初始化字典存数字和索引
    
    # 遍历数组，同时获取索引和值
    for i, num in enumerate(nums):  # 对应口诀2：遍历数组取索引与数值
        # 计算当前数字需要的补数
        complement = target - num  # 对应口诀3：计算当前数的补数
        
        # 如果补数在字典中，说明找到了答案
        if complement in num_map:
            return [num_map[complement], i]  # 对应口诀4：补数在字典则返回双索引
        
        # 如果没找到，就把当前数字和索引存入字典
        num_map[num] = i  # 对应口诀5：补数不在则存当前数和索引
    
    # 根据题目描述，输入保证有解，所以这里不会执行
    return []  # 对应口诀6：最终返回结果列表
```

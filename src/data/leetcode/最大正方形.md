# 最大正方形

## 【问题】
请解释 最大正方形 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/maximal-square/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772776499696-8ccdcd3d-c022-43e9-ad69-2de8d66dc1ae.png)

## 【回答】
**口诀速记：**

用动态规划 dp[i][j] 记录以 (i,j) 为右下角的最大全 1 正方形边长，通过取左、上、左上三个方向的最小边长再加 1，递推得到当前值，最终用最大边长的平方作为答案。

1. 空矩阵直接返 0
2. 初始化 DP 多行列
3. 遍历遇 1 取三小加 1
4. 更新最大边长
5. 返边长平方结果

**解法1：**

```python
def maximalSquare(matrix):
    if not matrix or not matrix[0]:
        return 0  # 对应口诀1：空矩阵直接返0（空矩阵边界处理）
    
    m = len(matrix)
    n = len(matrix[0])
    max_side = 0
    # 初始化DP表，多一行一列方便处理边界
    dp = [[0] * (n + 1) for _ in range(m + 1)]  # 对应口诀2：初始化DP多行列
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if matrix[i-1][j-1] == '1':
                # 对应口诀3：遍历遇1取三小加1（取左上/上/左最小值+1）
                dp[i][j] = min(dp[i-1][j], dp[i-1][j-1], dp[i][j-1]) + 1
                if dp[i][j] > max_side:
                    max_side = dp[i][j]  # 对应口诀4：更新最大边长
    
    return max_side * max_side  # 对应口诀5：返边长平方结果
```

**解法2：**

```python
def maximalSquare(matrix):
    if not matrix or not matrix[0]:
        return 0
    
    m = len(matrix)
    n = len(matrix[0])
    max_side = 0
    # 一维DP数组
    dp = [0] * (n + 1)
    prev = 0  # 用于保存 dp[i-1][j-1] 的值
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            temp = dp[j]  # 临时保存当前 dp[j]，它将作为下一轮的 prev
            if matrix[i-1][j-1] == '1':
                dp[j] = min(dp[j], dp[j-1], prev) + 1
                max_side = max(max_side, dp[j])
            else:
                dp[j] = 0
            prev = temp  # 更新 prev
    
    return max_side * max_side
```

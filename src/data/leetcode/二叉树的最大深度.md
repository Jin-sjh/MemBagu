# 二叉树的最大深度

## 【问题】
请解释 二叉树的最大深度 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/maximum-depth-of-binary-tree/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772767439557-4717d374-2619-494a-9508-006982c1ad5e.png)

## 【回答】
**口诀速记：**

通过递归（DFS）或层序遍历（BFS），计算二叉树中从根节点到最远叶子节点的路径上的节点总数。

- 递归终止条件：若当前节点为空，返回深度 0（空节点无深度）
- 递归计算当前节点的左子树深度、右子树深度
- 当前节点的深度 = 左右子树深度的最大值 + 1（+1 是当前节点本身的层级）
- 从根节点开始递归，最终返回的结果即为二叉树的最大深度

**解法1：**

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def maxDepth(self, root: TreeNode) -> int:
        # 递归终止条件：空节点深度为 0
        if not root:
            return 0  # 对应口诀1：空节点返回深度0
        # 递归计算左右子树的深度
        left_depth = self.maxDepth(root.left)  # 对应口诀2：计算左子树深度
        right_depth = self.maxDepth(root.right)  # 对应口诀2：计算右子树深度
        # 当前节点的深度 = 左右子树深度的最大值 + 1
        return max(left_depth, right_depth) + 1  # 对应口诀3：计算当前节点深度并返回
```

**解法2：**

```python
from collections import deque

class Solution:
    def maxDepth(self, root: TreeNode) -> int:
        if not root:
            return 0
        
        # 初始化队列，将根节点入队
        queue = deque([root])
        depth = 0
        
        while queue:
            # 记录当前层的节点数
            level_size = len(queue)
            # 遍历当前层的所有节点
            for _ in range(level_size):
                node = queue.popleft()
                # 将下一层的节点入队
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
            # 每遍历完一层，深度加 1
            depth += 1
        
        return depth
```

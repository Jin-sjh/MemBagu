# 子集

## 【问题】
请解释 子集 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/subsets/

## 【回答】
**口诀速记：**

通过 “逐步扩展” 或 “位掩码枚举” 的方式，系统地生成所有可能的元素组合，从而得到数组的全部子集。

- 初始化结果列表为包含空子集的列表 [[]]
- 遍历数组中的每一个数字：
- 对结果列表中已有的每一个子集，拼接当前数字生成新子集
- 将所有新子集添加到结果列表中（原子集 + 新子集）
- 遍历结束后返回结果列表（包含所有子集

**代码实现：**

```python
def subsets(nums):
    res = [[]]  # 对应口诀1：初始化结果列表（包含空子集）
    for num in nums:  # 对应口诀2：遍历每个数字
        res += [curr + [num] for curr in res]  # 对应口诀2：生成新子集并合并
    return res  # 对应口诀3：返回所有子集
```

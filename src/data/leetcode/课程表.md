# 课程表

## 【问题】
请解释 课程表 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/course-schedule/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772776101683-1e889ec6-90fe-4b8e-977e-60ef4d3b42ce.png)

## 【回答】
**口诀速记：**

将课程与先修关系抽象为有向图，通过拓扑排序（Kahn 算法或 DFS）判断图中是否存在环，无环则可完成所有课程，有环则不可。

- 建邻接表记入度
- 入度 0 节点入队列
- 出队计数减邻度
- 度 0 入队循环完
- 计数等课数返回

**代码实现：**

```python
from collections import deque

def canFinish(numCourses, prerequisites):
    # 1. 构建邻接表和入度数组
    adj = [[] for _ in range(numCourses)]  # 对应口诀1：建邻接表记入度（建邻接表）
    in_degree = [0] * numCourses           # 对应口诀1：建邻接表记入度（记入度数组）
    
    for course, pre in prerequisites:
        adj[pre].append(course)            # 对应口诀1：建邻接表记入度（填充邻接表）
        in_degree[course] += 1             # 对应口诀1：建邻接表记入度（更新入度）
    
    # 2. 初始化队列，加入所有入度为0的节点
    q = deque()                            # 对应口诀2：入度0节点入队列（初始化队列）
    for i in range(numCourses):
        if in_degree[i] == 0:
            q.append(i)                    # 对应口诀2：入度0节点入队列（入度0入队）
    
    # 3. 拓扑排序
    count = 0
    while q:
        u = q.popleft()                    # 对应口诀3：出队计数减邻度（出队）
        count += 1                         # 对应口诀3：出队计数减邻度（计数）
        # 遍历u的所有后继节点v
        for v in adj[u]:
            in_degree[v] -= 1              # 对应口诀3：出队计数减邻度（减邻度）
            if in_degree[v] == 0:
                q.append(v)                # 对应口诀4：度0入队循环完（度0入队）
    
    # 4. 判断是否所有课程都被访问到
    return count == numCourses             # 对应口诀5：计数等课数返回
```

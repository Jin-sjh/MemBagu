# 在排序数组中查找元素的第一个和最后一个位置

## 【问题】
请解释 在排序数组中查找元素的第一个和最后一个位置 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772712732883-0a96aaec-f283-4f67-87b7-5afc2f798bd0.png)

## 【回答】
**口诀速记：**

通过两次二分查找，分别定位目标值在排序数组中的左边界和右边界，从而得到其首次和末次出现的位置。

- 定义找左边界函数：
- 初始化左右指针，左≤右时循环算 mid
- nums [mid]\<target 则左 = mid+1，否则右 = mid-1
- 循环结束检查左指针有效且值为 target，返回左指针，否则 - 1
- 定义找右边界函数：
- 初始化左右指针，左≤右时循环算 mid
- nums [mid]≤target 则左 = mid+1，否则右 = mid-1
- 循环结束检查右指针有效且值为 target，返回右指针，否则 - 1
- 调用左右边界函数，左边界非 - 1 则返回 [左，右]，否则 [-1,-1]

**代码实现：**

```python
def searchRange(nums, target):
    def find_left(nums, target):
        left, right = 0, len(nums) - 1
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] < target:
                left = mid + 1
            else:
                right = mid - 1  # 对应口诀1：左边界函数的二分逻辑
        # 检查左边界是否有效
        if left < len(nums) and nums[left] == target:
            return left
        return -1  # 对应口诀1：左边界有效性检查

    def find_right(nums, target):
        left, right = 0, len(nums) - 1
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] <= target:
                left = mid + 1
            else:
                right = mid - 1  # 对应口诀2：右边界函数的二分逻辑
        # 检查右边界是否有效
        if right >= 0 and nums[right] == target:
            return right
        return -1  # 对应口诀2：右边界有效性检查

    left_pos = find_left(nums, target)
    right_pos = find_right(nums, target)
    return [left_pos, right_pos] if left_pos != -1 else [-1, -1]  # 对应口诀3：结果整合返回
```

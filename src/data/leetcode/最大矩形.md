# 最大矩形

## 【问题】
请解释 最大矩形 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/maximal-rectangle/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772765907937-83479fab-2b50-45d1-b681-86344fdaffcf.png)

## 【回答】
**口诀速记：**

- 空矩阵 / 首行空直接返回 0，记录矩阵行数 rows、列数 cols
- 初始化高度数组 heights（长度 cols，全 0）、最大面积 max\_area=0
- 定义单调栈版最大矩形面积函数（带哨兵简化边界）：
- 栈初始化空， heights 前后加 0（哨兵）
- 遍历哨兵版 heights，栈非空且当前高度 < 栈顶高度时：✅ 弹出栈顶，计算高度 h = 弹出值，宽度 w = 当前索引 - 新栈顶 - 1✅ 更新该函数内的最大面积
- 否则将当前索引入栈，最终返回函数内最大面积
- 遍历矩阵每一行：
- 遍历每行的列 j：若当前元素为 '1' 则 heights [j]+1，否则重置为 0（构建柱状图）
- 调用面积函数计算当前柱状图的最大面积，更新全局 max\_area
- 遍历结束返回全局 max\_area

**代码实现：**

```python
def maximalRectangle(matrix):
    if not matrix or not matrix[0]:
        return 0  # 对应口诀1：空矩阵返回0
    
    rows = len(matrix)
    cols = len(matrix[0])
    heights = [0] * cols  # 对应口诀2：初始化高度数组
    max_area = 0  # 对应口诀2：初始化最大面积
    
    def largestRectangleArea(heights):
        stack = []
        max_area = 0
        heights = [0] + heights + [0]  # 哨兵，简化边界处理  # 对应口诀3
        for i in range(len(heights)):
            while stack and heights[i] < heights[stack[-1]]:
                h = heights[stack.pop()]
                w = i - stack[-1] - 1  # 对应口诀3：计算宽高
                max_area = max(max_area, h * w)  # 对应口诀3：更新面积
            stack.append(i)  # 对应口诀3：索引入栈
        return max_area  # 对应口诀3：返回函数内最大面积
    
    for row in matrix:  # 对应口诀4：遍历每一行
        for j in range(cols):
            if row[j] == '1':
                heights[j] += 1
            else:
                heights[j] = 0  # 对应口诀4：构建柱状图高度
        current_max = largestRectangleArea(heights)  # 对应口诀4：计算当前行柱状图面积
        max_area = max(max_area, current_max)  # 对应口诀4：更新全局最大面积
    
    return max_area  # 对应口诀5：返回最终结果
```

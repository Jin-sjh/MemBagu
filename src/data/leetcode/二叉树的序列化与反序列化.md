# 二叉树的序列化与反序列化

## 【问题】
请解释 二叉树的序列化与反序列化 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/serialize-and-deserialize-binary-tree/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772777859999-ad7618e2-cfad-4c48-942e-0fce5eb270de.png)

## 【回答】
**口诀速记：**

通过一种遍历方式（如前序遍历或层序遍历）完整记录二叉树的节点值及空节点信息，再依据相同遍历逻辑反向重建出原树结构。

#### 序列化（encode）

1. 空树返空括号
2. 队列存根层序遍
3. 有值存值无存 null
4. 去尾 null 拼字符串

#### 反序列化（decode）

1. 空串直接返空树
2. 解析字符串建根
3. 队列存根逐节点
4. 按序建左右子树

**解法1：**

```python
# Definition for a binary tree node.
# class TreeNode(object):
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Codec:
    # 序列化：二叉树 -> 字符串
    def serialize(self, root):
        """
        将二叉树序列化为字符串，空节点用 '#' 表示，节点间用 ',' 分隔
        :type root: TreeNode
        :rtype: str
        """
        # 递归终止条件：空节点返回 '#'
        if not root:
            return '#'
        
        # 前序遍历：根节点 -> 左子树 -> 右子树
        left_str = self.serialize(root.left)
        right_str = self.serialize(root.right)
        
        # 拼接成字符串，用逗号分隔
        return f"{root.val},{left_str},{right_str}"

    # 反序列化：字符串 -> 二叉树
    def deserialize(self, data):
        """
        将序列化的字符串还原为二叉树
        :type data: str
        :rtype: TreeNode
        """
        # 将字符串按逗号分割成列表，用迭代器处理（方便逐个取元素）
        nodes = iter(data.split(','))
        
        # 递归构建二叉树
        def build_tree():
            # 取当前节点值
            val = next(nodes)
            # 如果是 '#'，表示空节点
            if val == '#':
                return None
            # 构建当前节点（根节点）
            node = TreeNode(int(val))
            # 构建左子树
            node.left = build_tree()
            # 构建右子树
            node.right = build_tree()
            return node
        
        return build_tree()

    # 注意最后要return node
```

**解法2：**

```python
class Codec:

    def serialize(self, root):
        """Encodes a tree to a single string.
        
        :type root: TreeNode
        :rtype: str
        """
        def dfs(node):
            if not node:
                return "null"
            return f"{node.val},{dfs(node.left)},{dfs(node.right)}"
        
        return dfs(root)
        

    def deserialize(self, data):
        """Decodes your encoded data to tree.
        
        :type data: str
        :rtype: TreeNode
        """
        from collections import deque
        data_list = deque(data.split(','))
        
        def dfs():
            val = data_list.popleft()
            if val == "null":
                return None
            node = TreeNode(int(val))
            node.left = dfs()
            node.right = dfs()
            return node
        
        return dfs()
```

# 回文链表

## 【问题】
请解释 回文链表 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/palindrome-linked-list/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772776827953-b10a5fe1-cfa8-49c3-82a6-f1dc869e3478.png)

## 【回答】
**口诀速记：**

通过快慢指针找到链表中点，反转后半段后与前半段逐一比对，从而在 O (1) 空间复杂度下判断链表是否为回文结构。

- 空单直接返真
- 快慢指针找中点
- 反转后半段链表
- 双指针逐值比对
- 全匹配返回真

**代码实现：**

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def isPalindrome(self, head: ListNode) -> bool:
        if not head or not head.next:
            return True  # 对应口诀1：空单直接返真（空/单节点链表是回文）
        
        # 1. 快慢指针找到中点
        slow = head
        fast = head
        while fast.next and fast.next.next:
            slow = slow.next  # 对应口诀2：快慢指针找中点（慢指针走一步）
            fast = fast.next.next  # 对应口诀2：快慢指针找中点（快指针走两步）
        
        # 2. 反转后半段链表
        def reverse(head):
            prev = None
            curr = head
            while curr:
                next_node = curr.next
                curr.next = prev
                prev = curr
                curr = next_node
            return prev
        
        right = reverse(slow.next)  # 对应口诀3：反转后半段链表（反转中点后链表）
        left = head
        
        # 3. 双指针比较
        while right:
            if left.val != right.val:
                return False  # 对应口诀4：双指针逐值比对（值不等返回假）
            left = left.next
            right = right.next
        
        return True  # 对应口诀5：全匹配返回真（所有值匹配返回真）
```

# 旋转图像

## 【问题】
请解释 旋转图像 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/rotate-image/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772724787720-401bbd30-c296-48d1-90c9-ce47d02fb831.png)

## 【回答】
**口诀速记：**

通过两次线性变换（转置 + 水平翻转，或上下翻转 + 转置），在 O (1) 额外空间内将 n×n 矩阵原地顺时针旋转 90 度

- 记录矩阵边长 n，先沿主对角线转置矩阵：
- 遍历行索引 i（0 到 n-1）
- 遍历列索引 j（i+1 到 n-1），交换 matrix [i][j] 和 matrix [j][i]
- 水平翻转每一行：
- 遍历每一行 i，初始化左右指针（左 0、右 n-1）
- 左 <右时交换 matrix [i][left] 和 matrix [i][right]，移动指针
- 全程原地修改矩阵，无返回值

**代码实现：**

```python
def rotate(matrix):
    n = len(matrix)
    # 1. 矩阵转置（沿主对角线）
    for i in range(n):
        for j in range(i + 1, n):
            matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]  # 对应口诀1：转置矩阵
    # 2. 水平翻转每一行
    for i in range(n):
        left, right = 0, n - 1  # 对应口诀2：初始化行内左右指针
        while left < right:
            matrix[i][left], matrix[i][right] = matrix[i][right], matrix[i][left]
            left += 1
            right -= 1  # 对应口诀2：水平翻转行
```

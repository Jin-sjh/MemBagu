# 任务调度器

## 【问题】
请解释 任务调度器 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/task-scheduler/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772782569468-c059c080-3a70-4bf5-a0da-b8b5cb72af04.png)

## 【回答】
**口诀速记：**

以出现次数最多的任务为框架，结合冷却时间构建调度模型，再用其他任务填充空闲时间，最终取理论最短时间与任务总数的较大值。

- 统计各任务出现频率
- 找出最大频率值
- 统计最大频率的任务数
- 算公式值与任务总数
- 取两者最大值返回

**代码实现：**

```python
from collections import Counter

def leastInterval(tasks, n):
    # 统计每个任务的频率
    task_counts = Counter(tasks)  # 对应口诀1：统计各任务出现频率
    # 找到最大频率
    max_count = max(task_counts.values())  # 对应口诀2：找出最大频率值
    # 找到有多少个任务达到了最大频率
    max_tasks = sum(1 for count in task_counts.values() if count == max_count)  # 对应口诀3：统计最大频率的任务数
    
    # 计算两种情况下的时间，取较大值
    return max((max_count - 1) * (n + 1) + max_tasks, len(tasks))  # 对应口诀4：算公式值与任务总数 + 口诀5：取两者最大值返回
```

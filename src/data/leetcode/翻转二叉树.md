# 翻转二叉树

## 【问题】
请解释 翻转二叉树 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/invert-binary-tree/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772776663891-87f84d31-db4b-41bb-9eab-f6fead362b20.png)

## 【回答】
**口诀速记：**

对二叉树的每个节点，递归或迭代地交换其左右子树，从而完成整棵树的镜像翻转。

- 空节点直接返
- 交换当前左右子
- 递归翻转左右树
- 最后返回根节点

**解法1：**

```python
class Solution:
    def invertTree(self, root: TreeNode) -> TreeNode:
        # 递归终止条件：空节点直接返回
        if not root:
            return None  # 对应口诀1：空节点直接返
        
        # 交换当前节点的左右子树
        root.left, root.right = root.right, root.left  # 对应口诀2：交换当前左右子
        
        # 递归翻转左子树
        self.invertTree(root.left)  # 对应口诀3：递归翻转左右树（翻左）
        # 递归翻转右子树
        self.invertTree(root.right)  # 对应口诀3：递归翻转左右树（翻右）
        
        return root  # 对应口诀4：最后返回根节点
```

**解法2：**

```python
from collections import deque

class Solution:
    def invertTree(self, root: TreeNode) -> TreeNode:
        if not root:
            return None
        
        # 初始化队列，加入根节点
        queue = deque([root])
        
        while queue:
            # 取出队首节点
            node = queue.popleft()
            # 交换其左右子树
            node.left, node.right = node.right, node.left
            
            # 若子节点非空，加入队列等待处理
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        
        return root
```

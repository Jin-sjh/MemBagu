# 不同的二叉搜索树

## 【问题】
请解释 不同的二叉搜索树 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/unique-binary-search-trees/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772766349093-7f7caf4f-d2f8-410b-bc02-579753b7e637.png)

## 【回答】
**口诀速记：**

利用动态规划，将 n 个节点的二叉搜索树计数问题，分解为选择不同根节点后，左右子树计数的乘积之和，本质是求解第 n 个卡特兰数。

- 初始化 DP 数组（长度 n+1，全 0），设置边界值：
- dp [0] = 1（空树的情况，作为乘法的基准）
- dp [1] = 1（只有 1 个节点时，仅 1 种二叉搜索树）
- 遍历节点数 i 从 2 到 n（计算 i 个节点的二叉搜索树数量）：
- 遍历每个可能的根节点 k（1 到 i）：✅ 以 k 为根时，左子树有 k-1 个节点，右子树有 i-k 个节点✅ 累加 dp [k-1] × dp [i-k] 到 dp [i]（左右子树组合数相乘）
- 遍历结束返回 dp [n]（n 个节点的二叉搜索树总数）

**代码实现：**

```python
def numTrees(n: int) -> int:
    dp = [0] * (n + 1)
    dp[0] = 1  # 空树  # 对应口诀1：边界值（空树）
    dp[1] = 1  # 只有一个节点  # 对应口诀1：边界值（1个节点）
    
    for i in range(2, n + 1):  # 对应口诀2：遍历2到n个节点
        for k in range(1, i + 1):  # 对应口诀2：遍历每个可能的根节点
            # 以k为根，左子树有k-1个节点，右子树有i-k个节点
            dp[i] += dp[k - 1] * dp[i - k]  # 对应口诀2：累加组合数
    
    return dp[n]  # 对应口诀3：返回n个节点的总数
```

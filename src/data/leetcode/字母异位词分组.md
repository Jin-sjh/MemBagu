# 字母异位词分组

## 【问题】
请解释 字母异位词分组 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/group-anagrams/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772725195611-ab00dfd6-3c34-4873-9bd0-d5634f42d104.png)

## 【回答】
**口诀速记：**

利用 “字母异位词排序后结果相同” 或 “字符计数结果相同” 的本质特征，将其作为哈希表的键，从而实现高效分组

- 导入 defaultdict，初始化字母异位词映射字典（值为列表）
- 遍历输入字符串列表中的每个字符串：
- 将字符串排序后拼接成新字符串，作为字典的键
- 把原字符串添加到该键对应的列表中
- 将字典的所有值转为列表返回（即所有异位词分组）

**解法1：**

```python
from collections import defaultdict
from typing import List

def groupAnagrams(strs: List[str]) -> List[List[str]]:
    # 使用 defaultdict 避免键不存在的 KeyError
    anagram_map = defaultdict(list)  # 对应口诀1：初始化异位词映射字典
    
    for s in strs:  # 对应口诀2：遍历每个字符串
        # 将字符串排序后作为键
        key = ''.join(sorted(s))  # 对应口诀2：排序生成键
        anagram_map[key].append(s)  # 对应口诀2：原字符串加入对应列表
    
    # 返回所有分组
    return list(anagram_map.values())  # 对应口诀3：返回分组列表

# 测试示例
strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
print(groupAnagrams(strs))
# 输出: [["bat"],["nat","tan"],["ate","eat","tea"]]
```

**解法2：**

```python
from collections import defaultdict
from typing import List

def groupAnagrams(strs: List[str]) -> List[List[str]]:
    anagram_map = defaultdict(list)
    
    for s in strs:
        # 初始化26个字母的计数数组
        count = [0] * 26
        for c in s:
            count[ord(c) - ord('a')] += 1
        # 将计数数组转为元组作为键
        key = tuple(count)
        anagram_map[key].append(s)
    
    return list(anagram_map.values())
```

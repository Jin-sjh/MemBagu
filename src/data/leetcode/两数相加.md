# 两数相加

## 【问题】
请解释 两数相加 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/add-two-numbers/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772707766671-4f133463-2c66-4255-b262-6585b7d1d092.png)

## 【回答】
**口诀速记：**

模拟手工加法，逐位相加并维护进位，同时统一处理链表长度差异和最终进位，生成逆序存储的结果链表。

- 建哑节点与当前指针，初始化进位为 0
- 遍历链表至全空且无进位
- 取两链表当前值（空则为 0）
- 算总和与新进位，建节点存个位值
- 移动各指针至下一位置
- 返回哑节点的下一个节点

**代码实现：**

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
        # 创建哑节点和当前指针
        dummy = ListNode()
        curr = dummy
        carry = 0  # 对应口诀1：建哑节点与当前指针，初始化进位为0
        
        # 遍历两个链表，直到都为空且无进位
        while l1 or l2 or carry:  # 对应口诀2：遍历链表至全空且无进位
            # 取出当前位的值，为空则为0
            val1 = l1.val if l1 else 0
            val2 = l2.val if l2 else 0  # 对应口诀3：取两链表当前值（空则为0）
            
            # 计算和与进位
            total = val1 + val2 + carry
            carry = total // 10
            curr.next = ListNode(total % 10)  # 对应口诀4：算总和与新进位，建节点存个位值
            
            # 移动指针
            curr = curr.next
            if l1:
                l1 = l1.next
            if l2:
                l2 = l2.next  # 对应口诀5：移动各指针至下一位置
        
        # 返回哑节点的下一个节点
        return dummy.next  # 对应口诀6：返回哑节点的下一个节点
```

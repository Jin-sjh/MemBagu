# 寻找两个正序数组的中位数

## 【问题】
请解释 寻找两个正序数组的中位数 的解题思路和实现方法。

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772708356822-5face1cf-9ad7-4098-80b7-b940d65029d8.png)

## 【回答】
**口诀速记：**

将 “找中位数” 转化为 “在两个有序数组中找第 k 小元素” 的问题，通过二分法每次排除不可能成为候选的一半元素，从而在 O (log (m+n)) 的时间复杂度内高效求解。

- 定义找第 k 小函数，确保短数组在前
- 短数组遍历完，直接取长数组第 k 个
- k 为 1 时取两数组当前指针最小值
- 计算比较位置（防越界），排除更小的部分
- 总长度奇数取中间第 k 小值
- 总长度偶数取中间两值求平均

**代码实现：**

```python
def findMedianSortedArrays(nums1, nums2):
    def get_kth(nums1, nums2, i, j, k):
        # 确保 nums1 是较短的数组，简化边界处理
        if len(nums1) - i > len(nums2) - j:
            return get_kth(nums2, nums1, j, i, k)  # 对应口诀1：定义找第k小函数，确保短数组在前
        
        # 如果 nums1 已经遍历完，直接从 nums2 中取
        if i == len(nums1):
            return nums2[j + k - 1]  # 对应口诀2：短数组遍历完，直接取长数组第k个
        
        # 如果 k 是 1，取两个数组当前指针的最小值
        if k == 1:
            return min(nums1[i], nums2[j])  # 对应口诀3：k为1时取两数组当前指针最小值
        
        # 计算本次要比较的位置，防止越界
        idx1 = i + min(k // 2, len(nums1) - i)
        idx2 = j + min(k // 2, len(nums2) - j)
        
        if nums1[idx1 - 1] < nums2[idx2 - 1]:
            # 排除 nums1 中从 i 到 idx1-1 的元素
            return get_kth(nums1, nums2, idx1, j, k - (idx1 - i))
        else:
            # 排除 nums2 中从 j 到 idx2-1 的元素
            return get_kth(nums1, nums2, i, idx2, k - (idx2 - j))  # 对应口诀4：计算比较位置（防越界），排除更小的部分
    
    m, n = len(nums1), len(nums2)
    total = m + n
    if total % 2 == 1:
        return get_kth(nums1, nums2, 0, 0, (total + 1) // 2)  # 对应口诀5：总长度奇数取中间第k小值
    else:
        mid1 = get_kth(nums1, nums2, 0, 0, total // 2)
        mid2 = get_kth(nums1, nums2, 0, 0, total // 2 + 1)
        return (mid1 + mid2) / 2.0  # 对应口诀6：总长度偶数取中间两值求平均

# 测试示例
print(findMedianSortedArrays([1, 3], [2]))    # 输出 2.0
print(findMedianSortedArrays([1, 2], [3, 4])) # 输出 2.5
```

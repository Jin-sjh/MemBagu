# 组合总和

## 【问题】
请解释 组合总和 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/combination-sum/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772713024528-58c03f60-c771-43f0-98aa-60c7e03b7ac0.png)

## 【回答】
**口诀速记：**

先对候选数组排序以支持剪枝优化，再通过回溯算法从指定起点递归选择元素（允许重复选当前元素），在和超过目标值时及时剪枝，最终收集所有和为目标值的组合。

- 初始化结果列表，候选数组排序（为剪枝铺垫）
- 定义回溯函数（参数：起始索引、当前路径、当前和）
- 回溯终止：当前和等于目标，路径拷贝加入结果
- 遍历从起始索引开始的候选数
- 剪枝：当前和 + 候选数 > 目标则终止遍历（排序后后续数更大）
- 选择：候选数加入路径，递归（起始索引不变，允许重复选）
- 回溯：移除路径最后一个数（撤销选择）
- 启动回溯（起始 0、空路径、和为 0），返回结果列表

**代码实现：**

```python
def combinationSum(candidates, target):
    res = []
    # 步骤1：排序，为剪枝做准备
    candidates.sort()
    n = len(candidates)  # 对应口诀1：初始化结果+排序
    
    # 步骤2：定义回溯函数
    def backtrack(start, path, current_sum):
        # 终止条件1：找到有效组合
        if current_sum == target:
            res.append(path.copy())
            return  # 对应口诀2：回溯终止逻辑
        
        # 遍历从start开始的元素（控制不重复组合）
        for i in range(start, n):  # 对应口诀4：遍历起始索引后的候选数
            num = candidates[i]
            # 剪枝：当前和+num超过目标，后续更大的数无需遍历
            if current_sum + num > target:
                break  # 对应口诀5：剪枝逻辑
            # 做出选择
            path.append(num)
            # 递归：因可重复选，下一次仍从i开始
            backtrack(i, path, current_sum + num)  # 对应口诀6：选择+递归
            # 回溯：撤销选择
            path.pop()  # 对应口诀7：回溯撤销选择
    
    # 初始调用：从索引0开始，空路径，和为0
    backtrack(0, [], 0)  # 对应口诀8：启动回溯
    return res  # 对应口诀8：返回结果列表
```

# 不同路径

## 【问题】
请解释 不同路径 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/unique-paths/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772761181311-770b6d0e-70c6-42bb-847b-f7f796584226.png)

## 【回答】
**口诀速记：**

将网格路径问题转化为组合数学问题，通过计算从总步数中选择特定方向步数的组合数，直接得出路径总数。

- 计算总步数 a = m + n - 2（从左上到右下需走的总步数）
- 计算较小步数 b = min (m-1, n-1)（需向下 / 向右走的最少步数，简化计算）
- 初始化结果为 1，遍历 1 到 b 的所有整数 i：
- 结果 = 结果 × (a - b + i) //i（逐步计算组合数，避免大数溢出）
- 遍历结束返回结果

**代码实现：**

```python
def uniquePaths(m: int, n: int) -> int:
    a = m + n - 2  # 对应口诀1：计算总步数
    b = min(m - 1, n - 1)  # 对应口诀2：计算较小步数
    res = 1  # 初始化结果
    for i in range(1, b + 1):  # 对应口诀3：遍历1到b
        res = res * (a - b + i) // i  # 对应口诀3：逐步计算组合数
    return res  # 对应口诀4：返回结果

# 测试示例
print(uniquePaths(3, 7))  # 输出: 28
print(uniquePaths(3, 2))  # 输出: 3
```

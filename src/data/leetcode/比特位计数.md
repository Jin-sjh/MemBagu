# 比特位计数

## 【问题】
请解释 比特位计数 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/counting-bits/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772778978348-56febcab-3bfc-493a-8a96-162e90565823.png)

## 【回答】
**口诀速记：**

利用二进制数的位运算规律，通过动态规划递推每个数的二进制中 1 的个数 —— 一个数的 1 的个数等于其右移一位后数的 1 的个数，加上其最低位是否为 1（即 ans[i] = ans[i >> 1] + (i & 1)），从而在 O (n) 时间复杂度内完成求解。

- 结果数组初始全 0
- 遍历数求 1 的个数
- 右移取整加末位 1
- 最终返回结果数组

**代码实现：**

```python
def countBits(n: int) -> list[int]:
    ans = [0] * (n + 1)  # 对应口诀1：结果数组初始全0
    for i in range(1, n + 1):  # 对应口诀2：遍历数求1的个数
        # 右移1位等价于整除2，i&1判断末位是否为1
        ans[i] = ans[i >> 1] + (i & 1)  # 对应口诀3：右移取整加末位1
    return ans  # 对应口诀4：最终返回结果数组
```

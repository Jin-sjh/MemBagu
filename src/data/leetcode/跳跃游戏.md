# 跳跃游戏

## 【问题】
请解释 跳跃游戏 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/jump-game/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772760508366-3f52c84d-8b77-49ec-a6fc-531e251dfaf1.png)

## 【回答】
**口诀速记：**

用贪心策略维护当前能到达的最远位置，在遍历中不断更新这个边界，只要能覆盖到数组末尾就成功，若中途出现无法到达的位置则失败。

- 初始化最远可达位置为 0，记录数组长度 n
- 遍历数组每个索引 i：
- 若 i 超过最远可达位置（当前位置无法到达），直接返回 False
- 更新最远可达位置（取当前值、i+nums [i] 的较大值）
- 若最远可达位置≥数组最后一个索引，直接返回 True
- 遍历结束后返回 True（所有位置可达，必然能到终点）

**代码实现：**

```python
def canJump(nums):
    max_reach = 0
    n = len(nums)  # 对应口诀1：初始化最远可达位置、记录数组长度
    for i in range(n):  # 对应口诀2：遍历每个索引
        if i > max_reach:  # 当前位置已经无法到达
            return False  # 对应口诀2：i超最远可达，返回False
        max_reach = max(max_reach, i + nums[i])  # 更新最远可达位置  # 对应口诀2：更新最远可达
        if max_reach >= n - 1:  # 已经可以到达终点
            return True  # 对应口诀2：可达终点，返回True
    return True  # 遍历完所有位置，说明可以到达  # 对应口诀3：遍历结束返回True
```

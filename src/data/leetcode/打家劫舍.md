# 打家劫舍

## 【问题】
请解释 打家劫舍 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/house-robber/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772775651008-683276f7-f88f-4149-b31b-f81994be5012.png)

## 【回答】
**口诀速记：**

通过动态规划，在每一步选择 “偷当前房屋（则不能偷前一间）” 或 “不偷当前房屋（则继承前一间的最优解）”，取两者最大值，从而递推得到全局最优解。

【

- 空单直接返数值
- 前二初始取最值
- 遍历取大更状态
- 最终返回前值完

】

**代码实现：**

```python
def rob(nums):
    if not nums:
        return 0  # 对应口诀1：空单直接返数值（空数组返回0）
    if len(nums) == 1:
        return nums[0]  # 对应口诀1：空单直接返数值（单元素返回自身）
    
    # 初始化：prev_prev 是到前前一间的最大金额，prev 是到前一间的最大金额
    prev_prev, prev = nums[0], max(nums[0], nums[1])  # 对应口诀2：前二初始取最值
    
    for i in range(2, len(nums)):  # 对应口诀3：遍历取大更状态（遍历起始）
        # 当前的最大金额 = max(不偷当前房屋(prev), 偷当前房屋(prev_prev + nums[i]))
        current = max(prev, prev_prev + nums[i])  # 对应口诀3：遍历取大更状态（取大）
        # 更新前前和前一间的状态
        prev_prev, prev = prev, current  # 对应口诀3：遍历取大更状态（更状态）
    
    return prev  # 对应口诀4：最终返回前值完
```

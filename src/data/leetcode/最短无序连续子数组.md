# 最短无序连续子数组

## 【问题】
请解释 最短无序连续子数组 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/shortest-unsorted-continuous-subarray/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772782263782-78f3d8f5-8db8-48d5-bd2f-4bb7c8b5bc38.png)

## 【回答】
**口诀速记：**

先通过两次遍历锁定无序子数组的初始边界，再根据边界内的最小/最大值向两侧扩展，最终确定能让整体有序的最小连续区间。

- 短数组直接返回 0
- 找左边界到降序处
- 全有序则返回 0
- 找右边界到升序处
- 求中间子数组最值
- 左扩至值≤最小值
- 右扩至值≥最大值
- 返回边界长度

**代码实现：**

```python
def findUnsortedSubarray(nums):
    n = len(nums)
    if n <= 1:
        return 0  # 对应口诀1：短数组直接返回0
    
    # 1. 找左边界 left
    left = 0
    while left < n - 1 and nums[left] <= nums[left + 1]:
        left += 1  # 对应口诀2：找左边界到降序处
    if left == n - 1:  # 整个数组已经有序
        return 0  # 对应口诀3：全有序则返回0
    
    # 2. 找右边界 right
    right = n - 1
    while right > 0 and nums[right] >= nums[right - 1]:
        right -= 1  # 对应口诀4：找右边界到升序处
    
    # 3. 找中间子数组的最小值和最大值
    min_val = min(nums[left:right+1])
    max_val = max(nums[left:right+1])  # 对应口诀5：求中间子数组最值
    
    # 4. 扩展左边界
    while left > 0 and nums[left - 1] > min_val:
        left -= 1  # 对应口诀6：左扩至值≤最小值
    
    # 5. 扩展右边界
    while right < n - 1 and nums[right + 1] < max_val:
        right += 1  # 对应口诀7：右扩至值≥最大值
    
    return right - left + 1  # 对应口诀8：返回边界长度
```

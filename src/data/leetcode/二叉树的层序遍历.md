# 二叉树的层序遍历

## 【问题】
请解释 二叉树的层序遍历 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/binary-tree-level-order-traversal/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772767204876-285f9f0c-f594-4238-87f6-0729306eeb89.png)

## 【回答】
**口诀速记：**

借助队列实现广度优先搜索（BFS），通过固定每一轮处理队列中当前层的所有节点数量，来区分并收集每一层的节点值，最终实现按层遍历二叉树。

- 若根节点为空，直接返回空列表（空树无层序遍历结果）
- 初始化队列（用 deque，初始包含根节点）、结果列表 res 为空
- 当队列非空时循环处理每一层：
- 记录当前层节点数level_size（队列长度），初始化当前层结果列表current_level
- 遍历当前层的level_size个节点：✅ 弹出队列头部节点，将节点值加入current_level✅ 若节点有左 / 右子节点，依次加入队列（为下一层遍历做准备）
- 将当前层结果current_level加入最终结果 res
- 循环结束后返回 res（按层存储的节点值列表）

**代码实现：**

```python
from collections import deque
from typing import Optional, List

# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []  # 对应口诀1：空树返回空列表
        
        # 初始化队列和结果列表
        q = deque([root])  # 对应口诀2：队列初始含根节点
        res = []  # 对应口诀2：初始化结果列表
        
        while q:  # 对应口诀3：队列非空则处理当前层
            # 当前层的节点数
            level_size = len(q)  # 对应口诀3：记录当前层节点数
            current_level = []  # 对应口诀3：初始化当前层结果
            
            # 遍历当前层的所有节点
            for _ in range(level_size):
                node = q.popleft()  # 对应口诀3：弹出队首节点
                current_level.append(node.val)  # 对应口诀3：节点值加入当前层
                
                # 将子节点加入队列
                if node.left:
                    q.append(node.left)  # 对应口诀3：左子节点入队
                if node.right:
                    q.append(node.right)  # 对应口诀3：右子节点入队
            
            # 将当前层的结果加入最终列表
            res.append(current_level)  # 对应口诀3：当前层结果入总列表
        
        return res  # 对应口诀4：返回层序遍历结果
```

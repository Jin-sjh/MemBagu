# 除法求值

## 【问题】
请解释 除法求值 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/evaluate-division/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772779445752-064e6155-7770-4340-a9d5-6838b21a09ee.png)

## 【回答】
**口诀速记：**

把变量和比值关系抽象成带权图，通过并查集（或 BFS/DFS）来维护和查询变量之间的比值关系。

#### 并查集（DSU）核心

1. 父节点存映射，权值记比值
2. 查找递归找根，路径压缩更权
3. 合并两节点，按比值连根更权
4. 查比值先判连通，连通则算权值比

#### 主逻辑

1. 初始化带权并查集
2. 遍历等式建集合并权
3. 遍历查询返比值或 - 1

**解法1：**

```python
from typing import List

class DSU:
    def __init__(self):
        self.parent = {}  # 存储每个节点的父节点
        self.weight = {}  # 存储 node / parent 的值
        # 对应并查集口诀1：父节点存映射，权值记比值

    def find(self, x):
        if x not in self.parent:
            # 如果节点不存在，初始化它指向自己，权值为1.0
            self.parent[x] = x
            self.weight[x] = 1.0
        if self.parent[x] != x:
            # 递归查找父节点，并进行路径压缩
            orig_parent = self.parent[x]
            self.parent[x] = self.find(self.parent[x])
            # 更新权值：x / 新父节点 = (x / 旧父节点) * (旧父节点 / 新父节点)
            self.weight[x] *= self.weight[orig_parent]
        return self.parent[x]
        # 对应并查集口诀2：查找递归找根，路径压缩更权

    def union(self, x, y, val):
        # 合并 x 和 y，满足 x / y = val
        root_x = self.find(x)
        root_y = self.find(y)
        if root_x != root_y:
            # 让 root_x 的父节点指向 root_y
            self.parent[root_x] = root_y
            # 计算权值：root_x / root_y = (x / y) / (x / root_x) * (y / root_y)
            self.weight[root_x] = val * self.weight[y] / self.weight[x]
        # 对应并查集口诀3：合并两节点，按比值连根更权

    def get_ratio(self, x, y):
        # 如果 x 或 y 不存在，或者不连通，返回 -1.0
        if x not in self.parent or y not in self.parent:
            return -1.0
        root_x = self.find(x)
        root_y = self.find(y)
        if root_x != root_y:
            return -1.0
        # x / y = (x / root) / (y / root)
        return self.weight[x] / self.weight[y]
        # 对应并查集口诀4：查比值先判连通，连通则算权值比

class Solution:
    def calcEquation(self, equations: List[List[str]], values: List[float], queries: List[List[str]]) -> List[float]:
        dsu = DSU()  # 对应主逻辑口诀1：初始化带权并查集
        # 构建并查集
        for (a, b), val in zip(equations, values):
            dsu.union(a, b, val)  # 对应主逻辑口诀2：遍历等式建集合并权
        # 处理查询
        return [dsu.get_ratio(c, d) for c, d in queries]  # 对应主逻辑口诀3：遍历查询返比值或-1
```

**解法2：**

```python
from collections import defaultdict, deque
from typing import List

class Solution:
    def calcEquation(self, equations: List[List[str]], values: List[float], queries: List[List[str]]) -> List[float]:
        # 构建图
        graph = defaultdict(list)
        for (a, b), val in zip(equations, values):
            graph[a].append((b, 1/val))  # a / b = val => b / a = 1/val
            graph[b].append((a, val))     # b / a = 1/val => a / b = val

        res = []
        for c, d in queries:
            # 如果 c 或 d 不在图中，直接返回 -1.0
            if c not in graph or d not in graph:
                res.append(-1.0)
                continue
            # BFS 查找从 d 到 c 的路径
            q = deque()
            q.append((d, 1.0))
            visited = set()
            visited.add(d)
            found = False
            while q:
                node, val = q.popleft()
                if node == c:
                    res.append(val)
                    found = True
                    break
                for neighbor, w in graph[node]:
                    if neighbor not in visited:
                        visited.add(neighbor)
                        q.append((neighbor, val * w))
            if not found:
                res.append(-1.0)
        return res
```

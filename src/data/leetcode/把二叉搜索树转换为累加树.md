# 把二叉搜索树转换为累加树

## 【问题】
请解释 把二叉搜索树转换为累加树 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/convert-bst-to-greater-tree/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772781376928-d7814fde-decb-4137-ab5d-14f5e9fe6830.png)

## 【回答】
**口诀速记：**

利用二叉搜索树的性质，通过\*\*反向中序遍历（右 - 根 - 左）\*\*，在遍历过程中维护一个累加和，用这个和直接更新每个节点的值。

- 初始化累加和为 0
- DFS 先遍历右子树
- 累加当前节点值到总和
- 更新节点值为总和
- 再遍历左子树
- 最终返回根节点

**代码实现：**

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution:
    def convertBST(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        self.sum = 0  # 对应口诀1：初始化累加和为0
        
        def dfs(node):
            if not node:
                return
            # 1. 先遍历右子树
            dfs(node.right)  # 对应口诀2：DFS先遍历右子树
            # 2. 处理当前节点：更新sum和节点值
            self.sum += node.val  # 对应口诀3：累加当前节点值到总和
            node.val = self.sum   # 对应口诀4：更新节点值为总和
            # 3. 再遍历左子树
            dfs(node.left)       # 对应口诀5：再遍历左子树
        
        dfs(root)
        return root  # 对应口诀6：最终返回根节点
```

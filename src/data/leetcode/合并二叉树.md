# 合并二叉树

## 【问题】
请解释 合并二叉树 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/merge-two-binary-trees/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772782445471-7c6d7bdd-db16-493c-9bed-25b2761d69e4.png)

## 【回答】
**口诀速记：**

通过深度优先搜索（DFS）递归遍历两棵二叉树的对应节点，将重叠节点的值相加，非重叠节点则直接复用，最终构建出合并后的新二叉树。

- 两节点均空返回空
- 一节点空返回另一节点
- 合并当前节点值建新节点
- 递归合并左右子树
- 最终返回合并后的节点

**代码实现：**

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def mergeTrees(self, root1: TreeNode, root2: TreeNode) -> TreeNode:
        # 如果两个节点都为空，返回空
        if not root1 and not root2:
            return None  # 对应口诀1：两节点均空返回空
        # 如果其中一个为空，直接返回另一个
        if not root1:
            return root2  # 对应口诀2：一节点空返回另一节点
        if not root2:
            return root1  # 对应口诀2：一节点空返回另一节点
        
        # 合并当前节点的值
        merged_node = TreeNode(root1.val + root2.val)  # 对应口诀3：合并当前节点值建新节点
        # 递归合并左子树和右子树
        merged_node.left = self.mergeTrees(root1.left, root2.left)
        merged_node.right = self.mergeTrees(root1.right, root2.right)  # 对应口诀4：递归合并左右子树
        
        return merged_node  # 对应口诀5：最终返回合并后的节点
```

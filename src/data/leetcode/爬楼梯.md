# 爬楼梯

## 【问题】
请解释 爬楼梯 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/climbing-stairs/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772761604350-7e39b235-e878-4e53-bbff-716d906d1cb8.png)

## 【回答】
**口诀速记：**

通过动态规划将问题分解为子问题，利用 “最后一步只能是从 n-1 阶爬 1 阶或从 n-2 阶爬 2 阶” 的递推关系，转化为斐波那契数列求解。

- 若 n 等于 1，直接返回 1（仅 1 种爬法：一步 1 阶）
- 初始化前两个状态：a=1（n=1 的爬法数）、b=2（n=2 的爬法数）
- 从 3 到 n 遍历（循环 n-2 次）：
- 交替更新 a 和 b：a 变为原 b，b 变为原 a + 原 b（状态转移）
- 遍历结束返回 b（即 n 阶的爬法数）

**代码实现：**

```python
def climbStairs(n: int) -> int:
    if n == 1:
        return 1  # 对应口诀1：n=1返回1
    a, b = 1, 2  # 对应口诀2：初始化前两个状态
    for _ in range(3, n + 1):  # 对应口诀3：遍历3到n
        a, b = b, a + b  # 对应口诀3：状态转移更新
    return b  # 对应口诀4：返回最终结果
```

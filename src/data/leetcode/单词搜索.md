# 单词搜索

## 【问题】
请解释 单词搜索 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/word-search/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772764219769-33ac980a-f2a4-4e58-86e7-ae4eb042fa40.png)

## 【回答】
**口诀速记：**

通过遍历网格找到单词首字符的所有可能起点，再利用深度优先搜索（DFS）向四个方向递归探索字符匹配路径，同时通过回溯（标记 / 恢复当前单元格）避免重复使用同一位置，最终验证是否存在完整匹配单词的路径。

- 空网格 / 首行空直接返回 False，记录网格行数 m、列数 n
- 定义 DFS 函数（参数：当前行 i、列 j、匹配字符索引 k）：
- 终止条件 1：k 等于单词长度（匹配完成），返回 True
- 终止条件 2：越界 / 字符不匹配，返回 False
- 标记当前位置为已访问（暂存字符，替换为占位符）
- 向上下左右四个方向递归 DFS（k+1），只要一个方向匹配成功则结果为 True
- 回溯：恢复当前位置的原字符，返回递归结果
- 遍历网格每个位置 (i,j)，调用 DFS (i,j,0)：
- 若任意位置 DFS 返回 True，直接返回 True
- 所有位置遍历完未匹配，返回 False

**代码实现：**

```python
def exist(board, word):
    if not board or not board[0]:
        return False  # 对应口诀1：空网格返回False
    
    m, n = len(board), len(board[0])  # 对应口诀1：记录行列数
    
    def dfs(i, j, k):
        # 已经匹配完所有字符
        if k == len(word):
            return True  # 对应口诀2：匹配完成返回True
        # 越界或字符不匹配
        if i < 0 or i >= m or j < 0 or j >= n or board[i][j] != word[k]:
            return False  # 对应口诀2：越界/不匹配返回False
        # 标记为已访问
        temp = board[i][j]
        board[i][j] = '#'  # 对应口诀2：标记已访问
        # 向四个方向搜索
        res = dfs(i+1, j, k+1) or dfs(i-1, j, k+1) or dfs(i, j+1, k+1) or dfs(i, j-1, k+1)  # 对应口诀2：四方向递归
        # 回溯，恢复原字符
        board[i][j] = temp  # 对应口诀2：回溯恢复
        return res  # 对应口诀2：返回递归结果
    
    for i in range(m):
        for j in range(n):
            if dfs(i, j, 0):
                return True  # 对应口诀3：任意位置匹配成功返回True
    return False  # 对应口诀4：全部遍历未匹配返回False
```

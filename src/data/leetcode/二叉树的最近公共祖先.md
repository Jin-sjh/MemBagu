# 二叉树的最近公共祖先

## 【问题】
请解释 二叉树的最近公共祖先 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772776954413-ca5ce74d-d286-4501-bd75-8165cae14714.png)

## 【回答】
**口诀速记：**

通过后序遍历递归地在左右子树中查找目标节点，若两节点分别在两侧则当前节点为 LCA，若在同一侧则返回该侧的查找结果。

- 空或目标返节点
- 递归查找左右子
- 左右都有返当前
- 否则返非空结果

**代码实现：**

```python
class TreeNode:
    def __init__(self, x):
        self.val = x
        self.left = None
        self.right = None

class Solution:
    def lowestCommonAncestor(self, root: 'TreeNode', p: 'TreeNode', q: 'TreeNode') -> 'TreeNode':
        # 递归终止条件：当前节点为空，或者当前节点就是p或q
        if not root or root == p or root == q:
            return root  # 对应口诀1：空或目标返节点
        
        # 递归查找左子树
        left = self.lowestCommonAncestor(root.left, p, q)  # 对应口诀2：递归查找左右子（查左）
        # 递归查找右子树
        right = self.lowestCommonAncestor(root.right, p, q)  # 对应口诀2：递归查找左右子（查右）
        
        # 如果左右子树都找到了节点，说明当前节点就是LCA
        if left and right:
            return root  # 对应口诀3：左右都有返当前
        # 否则，返回非空的那个子树的结果
        return left if left else right  # 对应口诀4：否则返非空结果
```

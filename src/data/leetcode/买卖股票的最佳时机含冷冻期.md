# 买卖股票的最佳时机含冷冻期

## 【问题】
请解释 买卖股票的最佳时机含冷冻期 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-with-cooldown/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772778308174-f8608756-da5b-49c2-aaba-ce8a79df8566.png)

## 【回答】
**口诀速记：**

通过动态规划，将每天的状态划分为「持有股票」「当天卖出」「不持有且非当天卖出」三类，用前一天的状态推导出当天的最优决策，从而在冷冻期约束下最大化利润。

- 短数组直接返 0
- 初始化持售休状态
- 遍历更持仓取最大
- 卖出仅由持仓转
- 休息取售休最大值
- 最终返售休最大值

**代码实现：**

```python
def maxProfit(prices):
    if len(prices) <= 1:
        return 0  # 对应口诀1：短数组直接返0
    
    prev_hold = -prices[0]    # 持仓状态
    prev_sold = float('-inf') # 卖出状态
    prev_rest = 0             # 休息/冷冻状态
    # 对应口诀2：初始化持售休状态
    
    for i in range(1, len(prices)):
        # 持仓：要么继续持仓，要么休息后买入
        curr_hold = max(prev_hold, prev_rest - prices[i])  # 对应口诀3：遍历更持仓取最大
        # 卖出：只能由持仓状态卖出
        curr_sold = prev_hold + prices[i]                 # 对应口诀4：卖出仅由持仓转
        # 休息：要么是之前卖出后休息，要么一直休息
        curr_rest = max(prev_sold, prev_rest)             # 对应口诀5：休息取售休最大值
        
        prev_hold, prev_sold, prev_rest = curr_hold, curr_sold, curr_rest
    
    # 最终利润是卖出或休息状态的最大值（持仓无利润）
    return max(prev_sold, prev_rest)  # 对应口诀6：最终返售休最大值
```

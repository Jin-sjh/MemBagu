# 最小栈

## 【问题】
请解释 最小栈 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/min-stack/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772774914120-eb5240cf-e4ad-44c7-a04a-c1eab60ff15e.png)

## 【回答】
**口诀速记：**

用\*\*双栈同步维护当前最小值\*\*，让辅助栈的栈顶始终是主栈的最小值，从而实现 O (1) 时间获取栈中最小元素。

双栈初配置，压值同步随； pop双栈出，top取栈尾； getMin查辅栈，最小值立回。

**代码实现：**

```python
class MinStack:

    def __init__(self):
        self.stack = []
        self.min_stack = []

    def push(self, val: int) -> None:
        self.stack.append(val)
        # 辅助栈压入当前最小值
        if not self.min_stack:
            self.min_stack.append(val)
        else:
            self.min_stack.append(min(val, self.min_stack[-1]))

    def pop(self) -> None:
        self.stack.pop()
        self.min_stack.pop()

    def top(self) -> int:
        return self.stack[-1]

    def getMin(self) -> int:
        return self.min_stack[-1]
```

# 盛最多水的容器

## 【问题】
请解释 盛最多水的容器 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/container-with-most-water/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772709987425-937e2600-c5fa-4df2-97c1-9d3b1ecfa76e.png)

## 【回答】
**口诀速记：**

用双指针从两端向中间收缩，每次移动高度较小的一侧指针，在一次遍历中找到能容纳最大水量的两条垂线

- 初始化左右指针（左 0、右为数组末尾），最大面积为 0
- 左指针 < 右指针时循环
- 算当前宽度（右 - 左）和高度（两指针值取最小）
- 计算当前面积，更新最大面积
- 移动较矮的指针（左矮右移左，否则右移右）
- 最终返回最大面积

**代码实现：**

```python
def maxArea(height):
    left = 0
    right = len(height) - 1
    max_area = 0  # 对应口诀1：初始化左右指针、最大面积
    
    while left < right:  # 对应口诀2：左指针<右指针时循环
        # 计算当前面积
        current_width = right - left
        current_height = min(height[left], height[right])  # 对应口诀3：算宽度和高度
        current_area = current_width * current_height
        
        # 更新最大面积
        if current_area > max_area:
            max_area = current_area  # 对应口诀4：更新最大面积
        
        # 移动指针
        if height[left] < height[right]:
            left += 1
        else:
            right -= 1  # 对应口诀5：移动较矮的指针
    
    return max_area  # 对应口诀6：返回最大面积
```

# 编辑距离

## 【问题】
请解释 编辑距离 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/edit-distance/

## 【回答】
**口诀速记：**

将两个字符串的整体转换问题拆解为前缀子串的转换子问题，通过比较当前字符是否相等，要么直接继承子问题结果，要么选择替换 / 删除 / 插入三种操作中代价最小的一种并累加操作次数，最终由子问题的最优解推导出原问题的最优解。

- 记录 word1 长度 m、word2 长度 n，创建 (m+1) 行 (n+1) 列的 DP 表（初始值全 0）
- 初始化边界：
- 第一列（word2 为空）：dp [i][0] = i（删除 word1 前 i 个字符的操作数）
- 第一行（word1 为空）：dp [0][j] = j（插入 word2 前 j 个字符的操作数）
- 填充 DP 表（遍历 i 从 1 到 m，j 从 1 到 n）：
- 若 word1 [i-1] == word2 [j-1]：无需操作，dp [i][j] = dp [i-1][j-1]
- 否则：取替换（dp [i-1][j-1]）、删除（dp [i-1][j]）、插入（dp [i][j-1]）的最小值 + 1
- 最终返回 dp [m][n]（word1 转 word2 的最小操作数）

**解法1：**

```python
def minDistance(word1: str, word2: str) -> int:
    m, n = len(word1), len(word2)
    
    # 创建DP表，dp[i][j]表示word1前i个字符转为word2前j个字符的最小操作数
    dp = [[0] * (n + 1) for _ in range(m + 1)]  # 对应口诀1：创建DP表
    
    # 初始化边界条件
    for i in range(m + 1):
        dp[i][0] = i  # 删除i次  # 对应口诀2：初始化第一列
    for j in range(n + 1):
        dp[0][j] = j  # 插入j次  # 对应口诀2：初始化第一行
    
    # 填充DP表
    for i in range(1, m + 1):
        for j in range(1, n + 1):  # 对应口诀3：遍历填充DP表
            if word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]  # 对应口诀3：字符相等无需操作
            else:
                dp[i][j] = min(dp[i-1][j-1], dp[i-1][j], dp[i][j-1]) + 1  # 对应口诀3：取最小操作+1
    
    return dp[m][n]  # 对应口诀4：返回最终结果
```

**解法2：**

```python
def minDistance(word1: str, word2: str) -> int:
    m, n = len(word1), len(word2)
    
    # 确保n是较小的，以优化空间
    if m < n:
        return minDistance(word2, word1)
    
    # 初始化一维DP数组
    dp = list(range(n + 1))
    
    for i in range(1, m + 1):
        # 记录左上角的值，即dp[i-1][j-1]
        prev = dp[0]
        # 更新dp[0]，即dp[i][0] = i
        dp[0] = i
        
        for j in range(1, n + 1):
            # 保存当前dp[j]的值，它将作为下一轮的prev（即dp[i][j]）
            temp = dp[j]
            if word1[i-1] == word2[j-1]:
                dp[j] = prev
            else:
                dp[j] = min(prev, dp[j], dp[j-1]) + 1
            # 更新prev为当前轮次的dp[j]，即下一轮的dp[i-1][j-1]
            prev = temp
    
    return dp[n]
```

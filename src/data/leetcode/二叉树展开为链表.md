# 二叉树展开为链表

## 【问题】
请解释 二叉树展开为链表 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772771014362-8040a4d3-7ed1-4939-9015-807891dbdd8a.png)

## 【回答】
**口诀速记：**

从二叉树最右下角的节点开始倒着往回处理（后序遍历先右后左），用 prev 记住上一个处理好的节点，把当前节点的左指针清空、右指针连上 prev，再把当前节点更新成新的 prev，一步步把整棵树拼成只有右指针的单链表。  （右取前，左置空，当更前）

*

- 初始化类属性prev为 None（记录上一个处理完成的节点）
- 定义后序遍历的 DFS 递归函数（参数：当前节点 node）：
- 终止条件：当前节点为空，直接返回
- 递归顺序：先遍历右子树，再遍历左子树（反常规后序，为了从右到左构建链表）
- 处理当前节点：✅ 右指针指向prev（将当前节点接入已处理的链表尾部）✅ 左指针置空（题目要求展开为单链表，无左子树）✅ 更新prev为当前节点（作为下一个节点的前驱）
- 调用 DFS 函数传入根节点 root，完成原地展开（无需返回值）

**代码实现：**

```python
# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def flatten(self, root: TreeNode) -> None:
        """
        Do not return anything, modify root in-place instead.
        """
        self.prev = None  # 记录上一个处理的节点  # 对应口诀1：初始化前驱节点
        
        def dfs(node):
            if not node:
                return  # 对应口诀2：终止条件（空节点返回）
            # 后序遍历：先右，再左，最后根
            dfs(node.right)  # 对应口诀2：递归右子树
            dfs(node.left)   # 对应口诀2：递归左子树
            
            # 处理当前节点
            node.right = self.prev  # 右指针指向前一个节点  # 对应口诀2：右指针接前驱
            node.left = None        # 左指针置空          # 对应口诀2：左指针置空
            self.prev = node        # 更新前一个节点为当前节点  # 对应口诀2：更新前驱
        
        dfs(root)  # 对应口诀3：调用DFS完成展开
```

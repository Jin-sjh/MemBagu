# 三数之和

## 【问题】
请解释 三数之和 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/3sum/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772710196330-490145c0-085d-4528-8122-ed3217f0e008.png)

## 【回答】
**口诀速记：**

先对数组排序，然后固定第一个数，用双指针在其右侧子数组中线性查找另外两个数，同时通过跳过重复元素来避免输出重复三元组。

- 数组排序，初始化结果列表
- 遍历每个数作为第一个数
- 第一个数 > 0 则直接终止循环（无解）
- 跳过重复的第一个数（i>0 且与前数相等）
- 初始化左指针（i+1）、右指针（数组末尾）
- 左 < 右时循环，计算三数之和
- 和 <0 左指针右移，和> 0 右指针左移
- 和 = 0 则加入结果，跳过重复的左右指针值
- 移动左右指针继续查找，最终返回结果

**代码实现：**

```python
def threeSum(nums):
    nums.sort()
    n = len(nums)
    res = []  # 对应口诀1：数组排序，初始化结果列表
    
    for i in range(n):  # 对应口诀2：遍历每个数作为第一个数
        # 如果当前数大于0，后面不可能有解
        if nums[i] > 0:
            break  # 对应口诀3：第一个数>0则终止循环
        # 跳过重复的第一个数
        if i > 0 and nums[i] == nums[i-1]:
            continue  # 对应口诀4：跳过重复的第一个数
        
        left = i + 1
        right = n - 1  # 对应口诀5：初始化左右指针
        
        while left < right:  # 对应口诀6：左<右时循环
            total = nums[i] + nums[left] + nums[right]  # 计算三数之和
            if total < 0:
                left += 1  # 对应口诀7：和<0左指针右移
            elif total > 0:
                right -= 1  # 对应口诀7：和>0右指针左移
            else:
                res.append([nums[i], nums[left], nums[right]])  # 对应口诀8：和=0加入结果
                # 跳过重复的第二个数
                while left < right and nums[left] == nums[left + 1]:
                    left += 1  # 对应口诀8：跳过重复左指针值
                # 跳过重复的第三个数
                while left < right and nums[right] == nums[right - 1]:
                    right -= 1  # 对应口诀8：跳过重复右指针值
                left += 1
                right -= 1  # 对应口诀9：移动左右指针
    return res  # 对应口诀9：最终返回结果
```

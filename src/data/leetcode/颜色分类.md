# 颜色分类

## 【问题】
请解释 颜色分类 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/sort-colors/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772762836651-9e6fa1bf-8746-48e4-b910-522bd95036dc.png)

## 【回答】
**口诀速记：**

用三指针一次遍历，将数组划分为 0、1、2 三个区域，通过交换元素实现原地排序。

- 初始化三个指针：left（0 的右边界）、curr（当前遍历指针）为 0，right（2 的左边界）为数组末尾
- 当 curr ≤ right 时循环遍历：
- 若 curr 位置值为 0：交换 left 和 curr 位置值，left、curr 均右移（0 归位）
- 若 curr 位置值为 1：仅 curr 右移（1 无需处理）
- 若 curr 位置值为 2：交换 curr 和 right 位置值，仅 right 左移（2 归位，curr 不移动需重新检查）
- 全程原地修改数组，无返回值

**代码实现：**

```python
def sortColors(nums):
    left = 0
    curr = 0
    right = len(nums) - 1  # 对应口诀1：初始化三个指针
    
    while curr <= right:  # 对应口诀2：curr≤right时循环
        if nums[curr] == 0:
            nums[left], nums[curr] = nums[curr], nums[left]
            left += 1
            curr += 1  # 对应口诀2：0归位，双指针右移
        elif nums[curr] == 1:
            curr += 1  # 对应口诀2：1无需处理，仅curr右移
        else:  # nums[curr] == 2
            nums[curr], nums[right] = nums[right], nums[curr]
            right -= 1  # 对应口诀2：2归位，仅right左移
```

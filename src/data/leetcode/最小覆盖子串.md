# 最小覆盖子串

## 【问题】
请解释 最小覆盖子串 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/minimum-window-substring/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772763518462-3f1f6b7d-3fe9-4d6b-9732-4413ff9e789a.png)

## 【回答】
**口诀速记：**

通过滑动窗口（双指针）动态调整窗口的左右边界，先扩展右边界找到包含目标字符串所有字符的窗口，再收缩左边界以找到满足条件的最小窗口，全程通过计数和校验机制确保窗口始终包含目标字符串的所有字符。

- 统计目标串 t 的字符需求（need 字典），记录需满足的字符种类数 need\_cnt
- 初始化窗口字符计数 window、满足需求的种类数 valid=0，左指针 left=0
- 初始化最小窗口长度（无穷大）、起始索引 start=0
- 右指针遍历 s 的每个字符：
- 若当前字符在 need 中：更新 window 计数，若计数等于 need 则 valid+1
- 当 valid=need\_cnt（窗口满足所有需求），进入收缩左边界循环：✅ 若当前窗口长度更小，更新最小窗口长度和起始索引✅ 移出左指针字符，若该字符在 need 中且计数刚好达标，valid-1，更新 window 计数✅ 左指针右移缩小窗口
- 最终若存在有效窗口，返回 s [start:start+min\_len]，否则返回空串

**代码实现：**

```python
def minWindow(s: str, t: str) -> str:
    from collections import defaultdict

    # 统计t中各字符的需求数量
    need = defaultdict(int)
    for c in t:
        need[c] += 1
    need_cnt = len(need)  # 需要满足的不同字符种类数  # 对应口诀1

    # 当前窗口内各字符的计数
    window = defaultdict(int)
    valid = 0  # 当前窗口中满足need条件的字符种类数  # 对应口诀2
    left = 0
    min_len = float('inf')
    start = 0  # 记录最小窗口的起始索引  # 对应口诀2、3

    for right in range(len(s)):  # 对应口诀4：右指针遍历
        c = s[right]
        if c in need:
            window[c] += 1
            if window[c] == need[c]:
                valid += 1  # 该字符满足需求，种类数+1  # 对应口诀4

        # 当窗口满足所有字符需求时，尝试收缩左边界
        while valid == need_cnt:  # 对应口诀4：满足需求则收缩
            # 更新最小窗口
            if right - left + 1 < min_len:
                min_len = right - left + 1
                start = left  # 对应口诀4：更新最小窗口

            # 移动左指针，缩小窗口
            d = s[left]
            left += 1
            if d in need:
                if window[d] == need[d]:
                    valid -= 1  # 该字符不再满足需求，种类数-1
                window[d] -= 1  # 对应口诀4：移出左字符并更新

    return s[start:start+min_len] if min_len != float('inf') else ""  # 对应口诀5
```

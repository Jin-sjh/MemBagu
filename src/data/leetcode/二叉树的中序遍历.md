# 二叉树的中序遍历

## 【问题】
请解释 二叉树的中序遍历 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/binary-tree-inorder-traversal/

## 【回答】
**口诀速记：**

按照 “左子树 → 根节点 → 右子树” 的顺序访问二叉树的所有节点，可以通过递归、栈迭代或 Morris 遍历等方式实现。

- 初始化结果列表 res 为空，定义递归 DFS 函数（参数：当前节点 node）：
- 终止条件：当前节点为空，直接返回
- 递归遍历左子树（先左）
- 将当前节点值加入结果列表（再根）
- 递归遍历右子树（后右）
- 调用 DFS 函数，传入根节点 root
- 遍历完成后返回结果列表 res（中序遍历结果）

**解法1：**

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        res = []  # 对应口诀1：初始化结果列表
        def dfs(node):
            if not node:
                return  # 对应口诀1：终止条件（节点空返回）
            dfs(node.left)   # 遍历左子树  # 对应口诀1：先左
            res.append(node.val)  # 访问根节点  # 对应口诀1：再根
            dfs(node.right)  # 遍历右子树  # 对应口诀1：后右
        dfs(root)  # 对应口诀2：调用DFS，传入根节点
        return res  # 对应口诀3：返回中序遍历结果
```

**解法2：**

```python
class Solution:
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        res = []
        stack = []
        current = root
        
        while current or stack:
            # 一直向左走到最左节点
            while current:
                stack.append(current)
                current = current.left
            # 弹出栈顶（最左节点），访问它
            current = stack.pop()
            res.append(current.val)
            # 转向右子树
            current = current.right
        return res
```

**解法3：**

```python
class Solution:
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        res = []
        current = root
        
        while current:
            if not current.left:
                # 无左子树，直接访问当前节点
                res.append(current.val)
                current = current.right
            else:
                # 找到左子树的最右节点（前驱节点）
                predecessor = current.left
                while predecessor.right and predecessor.right != current:
                    predecessor = predecessor.right
                
                if not predecessor.right:
                    # 建立临时线索，指向当前节点
                    predecessor.right = current
                    current = current.left
                else:
                    # 线索已存在，说明左子树已遍历完毕
                    predecessor.right = None  # 恢复树结构
                    res.append(current.val)
                    current = current.right
        return res
```

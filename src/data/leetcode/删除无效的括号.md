# 删除无效的括号

## 【问题】
请解释 删除无效的括号 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/remove-invalid-parentheses/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772778116126-f19f07d8-8cc1-4b12-a4cd-97e2c81ab660.png)

## 【回答】
**口诀速记：**

用广度优先搜索（BFS）按 “删除括号的数量” 分层遍历，在保证删除次数最少的前提下，找到所有有效的括号字符串。

- 验有效靠计数法
- 队列存串记已访
- 层序遍历查有效
- 有效则存标找到
- 未找到删括号扩队列
- 找到即停返结果

**代码实现：**

```python
from typing import List

class Solution:
    def removeInvalidParentheses(self, s: str) -> List[str]:
        # 存储最终结果的列表
        res = []
        
        # 第一步：统计需要移除的最少左括号(lremove)和右括号(rremove)数量
        lremove, rremove = 0, 0
        for c in s:
            if c == '(':
                # 遇到左括号，需要匹配的右括号数+1
                lremove += 1
            elif c == ')':
                # 遇到右括号
                if lremove == 0:
                    # 没有可匹配的左括号，这个右括号必须移除
                    rremove += 1
                else:
                    # 有可匹配的左括号，抵消一个需要匹配的左括号
                    lremove -= 1
        # 循环结束后：
        # lremove = 需要移除的左括号数量
        # rremove = 需要移除的右括号数量

        # 辅助函数：判断一个字符串的括号是否有效
        def isValid(str):
            cnt = 0  # 记录左括号减右括号的差值
            for c in str:
                if c == '(':
                    cnt += 1
                elif c == ')':
                    cnt -= 1
                    # 如果差值为负，说明右括号多于左括号，直接无效
                    if cnt < 0:
                        return False
            # 最终差值为0，说明括号完全匹配
            return cnt == 0

        # 回溯递归函数：尝试移除括号，寻找有效字符串
        # s: 当前处理的字符串
        # start: 开始遍历的索引（避免重复处理）
        # lremove: 还需要移除的左括号数量
        # rremove: 还需要移除的右括号数量
        def helper(s, start, lremove, rremove):
            # 递归终止条件：需要移除的括号都移除完了
            if lremove == 0 and rremove == 0:
                # 检查当前字符串是否有效，有效则加入结果集
                if isValid(s):
                    res.append(s)
                return

            # 从start位置开始遍历字符串，尝试移除括号
            for i in range(start, len(s)):
                # 去重优化：如果当前字符和前一个字符相同，且不是起始位置，跳过（避免重复结果）
                if i > start and s[i] == s[i - 1]:
                    continue
                
                # 剪枝优化：如果剩余字符数 < 还需要移除的括号数，不可能满足条件，直接终止循环
                if lremove + rremove > len(s) - i:
                    break
                
                # 情况1：尝试移除当前左括号（仅当还需要移除左括号时）
                if lremove > 0 and s[i] == '(':
                    # 移除s[i]位置的左括号，递归处理新字符串
                    # 下一次从i开始（因为移除后后面的字符前移了一位）
                    helper(s[:i] + s[i+1:], i, lremove - 1, rremove)
                
                # 情况2：尝试移除当前右括号（仅当还需要移除右括号时）
                if rremove > 0 and s[i] == ')':
                    # 移除s[i]位置的右括号，递归处理新字符串
                    helper(s[:i] + s[i+1:], i, lremove, rremove - 1)

        # 调用回溯函数，从索引0开始，传入初始需要移除的括号数量
        helper(s, 0, lremove, rremove)
        
        # 如果结果集为空（比如输入全是无效括号），返回空字符串
        return res if res else [""]
```

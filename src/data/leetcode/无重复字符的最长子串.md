# 无重复字符的最长子串

## 【问题】
请解释 无重复字符的最长子串 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/longest-substring-without-repeating-characters/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772708197647-f97b456e-a100-4ed3-9038-b261894f5450.png)

## 【回答】
**口诀速记：**

用滑动窗口（双指针）配合哈希表，在一次线性遍历中动态维护一个无重复字符的窗口，并记录窗口的最大长度。

- 初始化字符索引字典、左指针和最大长度为 0
- 遍历字符串取右指针与字符
- 字符在字典且索引≥左指针，左指针移至该索引 + 1
- 更新字符最后出现的索引为当前右指针
- 计算当前窗口长度，更新最大长度
- 最终返回最大长度

**代码实现：**

```python
def lengthOfLongestSubstring(s: str) -> int:
    char_map = {}  # 存储字符最后一次出现的索引
    left = 0
    max_len = 0  # 对应口诀1：初始化字符索引字典、左指针和最大长度为0
    
    for right, char in enumerate(s):  # 对应口诀2：遍历字符串取右指针与字符
        # 如果字符在map中，且其上次出现的位置 >= left，则移动左指针
        if char in char_map and char_map[char] >= left:
            left = char_map[char] + 1  # 对应口诀3：字符在字典且索引≥左指针，左指针移至该索引+1
        # 更新该字符最后一次出现的位置
        char_map[char] = right  # 对应口诀4：更新字符最后出现的索引为当前右指针
        # 计算当前窗口长度并更新最大值
        current_len = right - left + 1
        if current_len > max_len:
            max_len = current_len  # 对应口诀5：计算当前窗口长度，更新最大长度
    return max_len  # 对应口诀6：最终返回最大长度
```

# 寻找重复数

## 【问题】
请解释 寻找重复数 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/find-the-duplicate-number/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772777757708-d8c8c11f-f254-4f5e-b29a-65d7608ca727.png)

## 【回答】
**口诀速记：**

将数组视为特殊链表，利用快慢指针（Floyd 判圈算法）找到链表中环的入口，该入口对应的数值即为唯一重复的整数。

<https://www.doubao.com/thread/w69c6e8d1dd2072c3>

- 快慢指针始同点
- 快慢移动找相遇
- 慢归起点同速移
- 相遇即得重复数

**代码实现：**

```python
def findDuplicate(nums):
    # 初始化快慢指针
    slow = nums[0]
    fast = nums[0]  # 对应口诀1：快慢指针始同点
    
    # 第一步：找到快慢指针的相遇点
    while True:
        slow = nums[slow]
        fast = nums[nums[fast]]  # 对应口诀2：快慢移动找相遇（快慢移动）
        if slow == fast:
            break  # 对应口诀2：快慢移动找相遇（找相遇）
    
    # 第二步：将慢指针重置到起点，然后两指针同速移动，相遇点即为答案
    slow = nums[0]  # 对应口诀3：慢归起点同速移（慢归起点）
    while slow != fast:
        slow = nums[slow]
        fast = nums[fast]  # 对应口诀3：慢归起点同速移（同速移）
    
    return slow  # 对应口诀4：相遇即得重复数
```

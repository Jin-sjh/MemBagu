# 下一个排列

## 【问题】
请解释 下一个排列 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/next-permutation/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772712160640-ab341ec3-bb3e-467a-85ce-97c8831752c5.png)

## 【回答】
**口诀速记：**

从后向前找到第一个能增大排列的位置，用该位置后方最小的更大值与之交换，再将交换位置后方的降序子数组反转为升序，以此得到字典序最小的更大排列；若整个数组已是最大排列，则直接反转得到最小排列。

- 从后向前找首个升序对（nums [i]\<nums [i+1]），i 初始为倒数第二个索引
- 找到 i 后，从后向前找首个比 nums [i] 大的数 nums [j]
- 交换 nums [i] 和 nums [j]
- 反转 i+1 到末尾的子数组（i=-1 则反转整个数组）
- 全程原地修改数组，无返回值

**代码实现：**

```python
def nextPermutation(nums):
    """
    Do not return anything, modify nums in-place instead.
    """
    # 1. 从后向前找第一个升序对 nums[i] < nums[i+1]
    i = len(nums) - 2
    while i >= 0 and nums[i] >= nums[i+1]:
        i -= 1  # 对应口诀1：找首个升序对
    
    if i >= 0:  # 如果找到了这样的 i
        # 2. 从后向前找第一个比 nums[i] 大的数 nums[j]
        j = len(nums) - 1
        while nums[j] <= nums[i]:
            j -= 1  # 对应口诀2：找首个更大的数
        # 3. 交换 nums[i] 和 nums[j]
        nums[i], nums[j] = nums[j], nums[i]  # 对应口诀3：交换i和j位置
    
    # 4. 反转 i 之后的子数组（如果 i=-1，则反转整个数组）
    left, right = i + 1, len(nums) - 1
    while left < right:
        nums[left], nums[right] = nums[right], nums[left]
        left += 1
        right -= 1  # 对应口诀4：反转i+1后的子数组
```

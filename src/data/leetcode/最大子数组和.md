# 最大子数组和

## 【问题】
请解释 最大子数组和 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/maximum-subarray/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772725492987-2034c0d2-6451-4f3d-8509-7eb9a0bf01e1.png)

## 【回答】
**口诀速记：**

在遍历数组时，对每个元素都做出 “是延续前一个子数组，还是自己重新开始” 的最优选择，从而在单次遍历中找到全局最大子数组和。

- 空数组直接返回 0，初始化当前最大、全局最大为数组首个元素
- 遍历数组从第二个元素开始的所有数：
- 计算当前最大：取 “当前数” 和 “当前最大 + 当前数” 的较大值（决定续接 / 重启子数组）
- 若当前最大 > 全局最大，更新全局最大
- 遍历结束返回全局最大

**代码实现：**

```python
def maxSubArray(nums):
    if not nums:
        return 0  # 对应口诀1：空数组返回0
    
    current_max = global_max = nums[0]  # 对应口诀1：初始化当前/全局最大
    
    for num in nums[1:]:  # 对应口诀2：遍历第二个元素起的所有数
        # 决定是开启新的子数组，还是延续旧的子数组
        current_max = max(num, current_max + num)  # 对应口诀2：计算当前最大
        # 更新全局最大值
        if current_max > global_max:
            global_max = current_max  # 对应口诀2：更新全局最大
            
    return global_max  # 对应口诀3：返回全局最大
```

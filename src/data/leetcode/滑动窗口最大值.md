# 滑动窗口最大值

## 【问题】
请解释 滑动窗口最大值 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/sliding-window-maximum/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772777216355-f99337a2-2334-4cb5-8fcb-2cc4f002654a.png)

## 【回答】
**口诀速记：**

通过维护一个存储数组索引的\*\*单调递减双端队列\*\*，在窗口滑动过程中，始终移除队列中小于当前元素的索引和滑出窗口的索引，让队列首元素始终对应当前窗口的最大值，从而以 O (n) 的时间复杂度高效求解。

- 空数组直接返空
- 队列存索引
- 队尾小则弹出加新
- 队首滑窗则移除
- 窗口成型存队首值

**代码实现：**

```python
from collections import deque

def maxSlidingWindow(nums, k):
    if not nums:
        return []  # 对应口诀1：空数组直接返空
    
    q = deque()  # 存储索引，对应元素单调递减  # 对应口诀2：队列存索引保递减
    res = []
    
    for i in range(len(nums)):
        # 1. 移除队尾所有小于当前元素的索引，保持队列单调递减
        while q and nums[i] >= nums[q[-1]]:
            q.pop()  # 对应口诀3：队尾小则弹出加新（弹出小值索引）
        # 2. 将当前索引加入队尾
        q.append(i)  # 对应口诀3：队尾小则弹出加新（加新索引）
        
        # 3. 移除队首已经滑出窗口的索引
        while q[0] <= i - k:
            q.popleft()  # 对应口诀4：队首滑窗则移除
        
        # 4. 当窗口形成（i >= k-1）时，队首即为当前窗口最大值
        if i >= k - 1:
            res.append(nums[q[0]])  # 对应口诀5：窗口成型存队首值
    
    return res
```

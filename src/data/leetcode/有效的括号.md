# 有效的括号

## 【问题】
请解释 有效的括号 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/valid-parentheses/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772711077798-e8d61d48-bae3-4134-ac30-49b671308e0f.png)

## 【回答】
**口诀速记：**

利用栈的后进先出特性，对左括号进行入栈暂存，遇到右括号时则与栈顶的最近左括号进行类型匹配，最终通过栈是否为空验证所有括号是否完全闭合。

- 初始化空栈，定义右→左括号的映射字典
- 遍历字符串中的每个字符
- 遇右括号：取栈顶（空则补 #），不匹配则返回假
- 遇左括号：直接压入栈中
- 遍历结束后，栈空则返回真（全匹配），否则假

**代码实现：**

```python
def isValid(s: str) -> bool:
    stack = []
    mapping = {')': '(', '}': '{', ']': '['}  # 对应口诀1：初始化栈和映射字典
    
    for char in s:  # 对应口诀2：遍历每个字符
        if char in mapping:
            # 遇到右括号，检查栈顶是否匹配
            top_element = stack.pop() if stack else '#'
            if mapping[char] != top_element:
                return False  # 对应口诀3：右括号不匹配返回假
        else:
            # 遇到左括号，压入栈
            stack.append(char)  # 对应口诀4：左括号入栈
    
    # 栈为空则所有括号都匹配成功
    return not stack  # 对应口诀5：栈空返回真，否则假
```

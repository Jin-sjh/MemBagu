# 最小路径和

## 【问题】
请解释 最小路径和 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/minimum-path-sum/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772761434589-1f892b3b-d347-46ba-9731-649d29d27e1d.png)

## 【回答】
**口诀速记：**

通过动态规划，利用 “每个位置的最小路径和仅由其上方或左方的最小路径和转移而来” 的状态转移关系，逐步推导出从左上角到右下角的全局最小路径和。

- 空网格 / 首行空直接返回 0，记录网格行数 m、列数 n
- 初始化第一行：从第 2 列开始，每个位置累加左侧位置值（仅能从左来）
- 初始化第一列：从第 2 行开始，每个位置累加上方位置值（仅能从上来）
- 填充其余位置：遍历第 2 行到最后一行、第 2 列到最后一列：
- 当前位置值 += 上方 / 左侧位置值的较小值（选最短路径）
- 最终返回网格右下角位置的值（即最小路径和）

**代码实现：**

```python
def minPathSum(grid):
    if not grid or not grid[0]:
        return 0  # 对应口诀1：空网格返回0
    
    m, n = len(grid), len(grid[0])  # 对应口诀1：记录行列数
    
    # 初始化第一行
    for j in range(1, n):
        grid[0][j] += grid[0][j-1]  # 对应口诀2：初始化第一行
    
    # 初始化第一列
    for i in range(1, m):
        grid[i][0] += grid[i-1][0]  # 对应口诀3：初始化第一列
    
    # 填充其他位置
    for i in range(1, m):
        for j in range(1, n):
            grid[i][j] += min(grid[i-1][j], grid[i][j-1])  # 对应口诀4：填充其余位置
    
    return grid[-1][-1]  # 对应口诀5：返回右下角值
```

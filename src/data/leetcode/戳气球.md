# 戳气球

## 【问题】
请解释 戳气球 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/burst-balloons/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772778558474-72ee7a81-0f2e-4638-bd1b-0fbc5f0f7d56.png)

## 【回答】
**口诀速记：**

将 “戳破气球” 的过程逆向思考为 “添加气球”，通过动态规划定义dp[i][j]表示戳破区间(i,j)内所有气球的最大收益，枚举区间内最后一个被戳破的气球k，利用nums[i] * nums[k] * nums[j]（此时k的左右边界固定为i和j）来转移状态，最终求得全局最优解。

- 首尾加 1 扩数组
- DP 表记区间最大币
- 按长枚举区间
- 枚举起点定终点
- 枚举最后戳的球
- 算值更新 DP 表
- 返首尾区间最大值

三个循环

【确定区间长度，确定区间左右边界，枚举最后戳破气球】

状态转移：当前状态还是戳破左半边、戳破右半边、戳破部分数组

**代码实现：**

```python
def maxCoins(nums):
    # 在首尾添加虚拟的1
    nums = [1] + nums + [1]  # 对应口诀1：首尾加1扩数组
    n = len(nums)
    # 创建dp表，dp[i][j]表示戳破(i, j)区间内所有气球的最大硬币
    dp = [[0] * n for _ in range(n)]  # 对应口诀2：DP表记区间最大币
    
    # 按区间长度从小到大枚举
    for length in range(2, n):  # 对应口诀3：按长枚举区间
        # 枚举区间起点i
        for i in range(n - length):  # 对应口诀4：枚举起点定终点（枚举起点）
            j = i + length  # 对应口诀4：枚举起点定终点（定终点）
            # 枚举最后一个被戳破的气球k
            for k in range(i + 1, j):  # 对应口诀5：枚举最后戳的球
                # 计算并更新最大硬币数
                dp[i][j] = max(dp[i][j], dp[i][k] + dp[k][j] + nums[i] * nums[k] * nums[j])  # 对应口诀6：算值更新DP表
    
    return dp[0][n-1]  # 对应口诀7：返首尾区间最大值

# 测试示例
print(maxCoins([3, 1, 5, 8]))  # 输出: 167
print(maxCoins([1, 5]))        # 输出: 10
```

# 合并区间

## 【问题】
请解释 合并区间 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/merge-intervals/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772760750975-ce5a25f6-aab0-44c3-9b30-6a7a3a0cc95d.png)

## 【回答】
**口诀速记：**

先按区间起点排序，再从左到右遍历，通过比较当前区间与已合并区间的终点来决定是直接加入还是合并更新，从而高效地将所有重叠区间合并为不重叠的区间集合。

- 按区间起始值对区间列表排序（lambda x: x [0]）
- 初始化合并结果列表为空
- 遍历每个区间：
- 若结果为空，或当前区间起始 > 最后一个合并区间的结束（无重叠）：直接将当前区间加入结果
- 否则（有重叠）：更新最后一个合并区间的结束值为 “自身结束值” 和 “当前区间结束值” 的较大值
- 遍历结束返回合并后的结果列表

**代码实现：**

```python
def merge(intervals):
    # 按区间的起始值排序
    intervals.sort(key=lambda x: x[0])  # 对应口诀1：按起始值排序
    
    merged = []  # 对应口诀2：初始化合并结果列表
    for interval in intervals:  # 对应口诀3：遍历每个区间
        # 如果结果为空，或者当前区间与最后一个合并区间不重叠
        if not merged or interval[0] > merged[-1][1]:
            merged.append(interval)  # 对应口诀3：无重叠则加入
        else:
            # 重叠则合并，更新最后一个区间的结束值
            merged[-1][1] = max(merged[-1][1], interval[1])  # 对应口诀3：重叠则合并
    
    return merged  # 对应口诀4：返回合并结果
```

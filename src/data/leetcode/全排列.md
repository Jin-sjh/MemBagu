# 全排列

## 【问题】
请解释 全排列 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/permutations/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772724585076-80238767-dfd0-4aa8-9fdb-11dd2567c723.png)

## 【回答】
**口诀速记：**

通过回溯法，在每一步选择一个未使用的数字加入当前排列，递归构建所有可能的排列，再通过回溯恢复状态以探索所有分支

- 记录数组长度，初始化结果列表、使用标记数组（全 False）
- 定义回溯函数（参数：当前排列路径）
- 回溯终止：路径长度等于数组长度，路径拷贝加入结果
- 遍历数组所有索引：
- 索引未被使用时：标记为已用，将对应数加入路径
- 递归调用回溯函数，继续构建排列
- 回溯：移除路径最后一个数，标记索引为未用
- 启动回溯（空路径），返回结果列表

**代码实现：**

```python
def permute(nums):
    n = len(nums)
    res = []
    used = [False] * n  # 对应口诀1：初始化长度、结果、使用标记
    
    def backtrack(path):
        if len(path) == n:
            res.append(path.copy())
            return  # 对应口诀2、3：回溯函数+终止逻辑
        
        for i in range(n):  # 对应口诀4：遍历所有索引
            if not used[i]:
                used[i] = True
                path.append(nums[i])  # 对应口诀4：标记使用+加入路径
                backtrack(path)  # 对应口诀4：递归构建排列
                path.pop()
                used[i] = False  # 对应口诀4：回溯（移除数+取消标记）
    
    backtrack([])  # 对应口诀5：启动回溯
    return res  # 对应口诀5：返回结果
```

# 移动零

## 【问题】
请解释 移动零 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/move-zeroes/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772777651844-16183dc3-d876-4d1a-be17-12093f7244c2.png)

## 【回答】
**口诀速记：**

用双指针在一次遍历中，将所有非零元素按原顺序 “移动” 到数组前端，再将剩余位置填充为零，从而在原地完成零元素的后移。

- 记非零位初始 0
- 遍历移非零填前
- 剩余位置全填 0

**代码实现：**

```python
def moveZeroes(nums):
    non_zero_pos = 0  # 记录下一个非零元素应该放置的位置  # 对应口诀1：记非零位初始0
    # 第一次遍历：将所有非零元素移动到数组前端
    for i in range(len(nums)):
        if nums[i] != 0:
            nums[non_zero_pos] = nums[i]  # 对应口诀2：遍历移非零填前（填非零）
            non_zero_pos += 1  # 对应口诀2：遍历移非零填前（更新位置）
    # 第二次遍历：将剩余位置填充为0
    for i in range(non_zero_pos, len(nums)):
        nums[i] = 0  # 对应口诀3：剩余位置全填0
```

# 根据身高重建队列

## 【问题】
请解释 根据身高重建队列 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/queue-reconstruction-by-height/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772779641589-8365881f-bfe2-482b-b879-d1ef8e891fc4.png)

## 【回答】
**口诀速记：**

- 身高降序 k 升序排序
- 初始化空队列
- 遍历按 k 插入队列
- 最终返回队列

**代码实现：**

```python
def reconstructQueue(people):
    # 按身高降序，k升序排序
    people.sort(key=lambda x: (-x[0], x[1]))  # 对应口诀1：身高降序k升序排序
    queue = []  # 对应口诀2：初始化空队列
    for p in people:
        # 插入到第k个位置
        queue.insert(p[1], p)  # 对应口诀3：遍历按k插入队列
    return queue  # 对应口诀4：最终返回队列
    
# 测试示例
people = [[7,0],[4,4],[7,1],[5,0],[6,1],[5,2]]
print(reconstructQueue(people))  # 输出: [[5,0],[7,0],[5,2],[6,1],[4,4],[7,1]]
```

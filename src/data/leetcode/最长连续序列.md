# 最长连续序列

## 【问题】
请解释 最长连续序列 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/longest-consecutive-sequence/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772772089996-93be836a-7658-4aa3-a8c0-a52ec294c068.png)

## 【回答】
**口诀速记：**

利用哈希集合去重并实现 O (1) 查找，只从每个连续序列的起点（即 x-1 不存在于集合中的数）开始向后遍历计数，从而在 O (n) 时间内找到最长连续序列的长度。【 去重找起点，向后数连续，取长即为解  】

- 将数组转换为集合num_set（去重 + O (1) 时间查找），初始化最大连续长度max_len为 0
- 遍历集合中的每个数字num：
- 判定是否为连续序列起点：若num - 1不在集合中（说明无前驱，是序列第一个数）
- 若是起点，初始化当前数字current_num = num、当前长度current_len = 1
- 循环向后查找连续数：只要current_num + 1在集合中，就更新current_num和current_len
- 用当前长度更新max_len（取两者最大值）
- 遍历结束后返回max_len（最长连续序列的长度）

**代码实现：**

```python
def longestConsecutive(nums):
    num_set = set(nums)  # 对应口诀1：数组转集合（去重+快速查找）
    max_len = 0          # 对应口诀1：初始化最大长度
    
    for num in num_set:  # 对应口诀2：遍历集合中的每个数字
        # 只有当 num 是序列的起点时，才开始计算
        if num - 1 not in num_set:  # 对应口诀2：判定是否为序列起点
            current_num = num
            current_len = 1         # 对应口诀2：初始化当前数字和长度
            
            # 向后查找连续的数
            while current_num + 1 in num_set:  # 对应口诀2：循环查找连续数
                current_num += 1
                current_len += 1               # 对应口诀2：更新当前数字和长度
            
            # 更新最大长度
            max_len = max(max_len, current_len)  # 对应口诀2：更新最大长度
    
    return max_len  # 对应口诀3：返回最长连续序列长度
```

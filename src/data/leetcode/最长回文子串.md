# 最长回文子串

## 【问题】
请解释 最长回文子串 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/longest-palindromic-substring/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772708499247-d4258b8b-760c-498a-81e1-a764d12ec6f1.png)

## 【回答】
**口诀速记：**

利用回文子串的最优子结构，通过 “短串推长串” 的方式递推判断，避免重复计算。

- 短字符串直接返回原串
- 初始化最大长度为 1、起始位置为 0
- 构建 n×n DP 数组，对角线设为真（单字符回文）
- 遍历子串长度从 2 到 n
- 遍历起始索引，计算结束索引（越界则终止）
- 首尾字符不等则 DP 值为假
- 长度 < 3 且首尾相等则 DP 值为真，否则继承内层 DP 值
- 回文且长度更大时更新最大长度和起始位置
- 最终返回最长回文子串

以每个字符（或每对相邻字符）为中心，向左右扩展，找到以该中心为对称轴的最长回文子串，最终在所有可能的回文子串中选出最长的一个

**解法1：**

```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        n = len(s)

        if n < 2:
            return s  # 对应口诀1：短字符串直接返回原串
        
        max_len = 1
        begin = 0  # 对应口诀2：初始化最大长度为1、起始位置为0

        dp = [[False]*n for _ in range(n)]  # 对应口诀3：构建n×n DP数组
        
        for i in range(n):
            dp[i][i] = True  # 对应口诀3：对角线设为真（单字符回文）
        
        for L in range(2, n+1):  # 对应口诀4：遍历子串长度从2到n
            for i in range(n):  # 对应口诀5：遍历起始索引
                j = L+i-1
                if j >= n:
                    break  # 对应口诀5：计算结束索引（越界则终止）
                
                if s[i] != s[j]:
                    dp[i][j] = False  # 对应口诀6：首尾字符不等则DP值为假
                
                else:
                    if L < 3:
                        dp[i][j] = True  # 对应口诀7：长度<3且首尾相等则DP值为真
                    else:
                        dp[i][j] = dp[i+1][j-1]  # 对应口诀7：否则继承内层DP值
                
                if dp[i][j] and L > max_len:
                    max_len = L
                    begin = i  # 对应口诀8：回文且长度更大时更新最大长度和起始位置
        return s[begin:begin+max_len]  # 对应口诀9：最终返回最长回文子串
```

**解法2：**

```python
def longestPalindrome(s: str) -> str:
    if not s:
        return ""
    
    start = 0
    max_len = 1
    
    def expand(left: int, right: int) -> None:
        nonlocal start, max_len
        while left >= 0 and right < len(s) and s[left] == s[right]:
            if right - left + 1 > max_len:
                max_len = right - left + 1
                start = left
            left -= 1
            right += 1
    
    for i in range(len(s)):
        # 奇数长度回文
        expand(i, i)
        # 偶数长度回文
        expand(i, i + 1)
    
    return s[start:start + max_len]
```

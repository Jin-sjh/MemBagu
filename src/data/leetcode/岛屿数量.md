# 岛屿数量

## 【问题】
请解释 岛屿数量 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/number-of-islands/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772775794223-24dfe247-6fe6-45b1-9b2c-0a97320570e8.png)

## 【回答】
**口诀速记：**

遍历网格，每发现一块未被访问的陆地（'1'），就通过深度优先搜索（DFS）或广度优先搜索（BFS）将其相连的所有陆地标记为已访问，每完成一次这样的标记过程，就代表发现了一个新岛屿。

【

- 空网返 0
- 行列计数初始化
- 遇陆计数加一搜
- 越界非陆直接返
- 标 0 再搜四方向
- 遍历完返总计数

】

**解法1：**

```python
def numIslands(grid):
    if not grid or not grid[0]:
        return 0  # 对应口诀1：空网直接返回0
    
    rows = len(grid)  # 对应口诀2：行列计数先初始化
    cols = len(grid[0])  # 对应口诀2：行列计数先初始化
    count = 0  # 对应口诀2：行列计数先初始化
    
    def dfs(i, j):
        # 越界或不是陆地，直接返回
        if i < 0 or i >= rows or j < 0 or j >= cols or grid[i][j] != '1':
            return  # 对应口诀4：深搜先判越界非陆
        # 将当前陆地标记为已访问（改为水）
        grid[i][j] = '0'  # 对应口诀5：标已访再搜四方向（标已访）
        # 递归探索四个方向
        dfs(i + 1, j)  # 对应口诀5：标已访再搜四方向（搜四方向）
        dfs(i - 1, j)
        dfs(i, j + 1)
        dfs(i, j - 1)
    
    # 遍历整个网格
    for i in range(rows):
        for j in range(cols):
            if grid[i][j] == '1':
                count += 1  # 对应口诀3：遇陆计数加一并深搜（计数加一）
                dfs(i, j)  # 对应口诀3：遇陆计数加一并深搜（并深搜）
    return count  # 对应口诀6：遍历完返回总计数
```

**解法2：**

```python
from collections import deque

def numIslands(grid):
    if not grid or not grid[0]:
        return 0
    
    rows = len(grid)
    cols = len(grid[0])
    count = 0
    
    for i in range(rows):
        for j in range(cols):
            if grid[i][j] == '1':
                count += 1
                # 将当前陆地标记为已访问
                grid[i][j] = '0'
                # 初始化队列，加入当前坐标
                q = deque()
                q.append((i, j))
                
                while q:
                    x, y = q.popleft()
                    # 检查四个方向
                    for dx, dy in [(-1, 0), (1, 0), (0, -1), (0, 1)]:
                        nx, ny = x + dx, y + dy
                        if 0 <= nx < rows and 0 <= ny < cols and grid[nx][ny] == '1':
                            grid[nx][ny] = '0'
                            q.append((nx, ny))
    return count
```

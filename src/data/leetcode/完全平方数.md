# 完全平方数

## 【问题】
请解释 完全平方数 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/perfect-squares/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772777533891-c92c6d1d-a294-4d1b-89a9-8c677be7f134.png)

## 【回答】
**口诀速记：**

利用四平方和定理，将问题的解空间从无限缩小到仅 4 种可能（1、2、3、4），然后通过快速数学判断，按优先级排除不可能的情况，从而在 O (√n) 时间内直接得出答案。

数学依据：这是四平方和定理的重要推论 ——一个数只能表示为 4 个平方数的和，当且仅当它可以写成 4ᵏ × (8m + 7) 的形式（k、m 为非负整数）。

- 本身平方直接返 1
- 4 幂 8 余 7 返 4
- 两平方和验证返 2
- 其余情况返回 3

**解法1：**

```python
import math

def numSquares(n: int) -> int:
    # 1. 判断是否本身就是完全平方数
    if int(math.isqrt(n)) ** 2 == n:
        return 1  # 对应口诀1：本身平方直接返1
    
    # 2. 判断是否满足 4^k * (8m + 7) 的形式
    while n % 4 == 0:
        n = n // 4
    if n % 8 == 7:
        return 4  # 对应口诀2：4幂8余7返4
    
    # 3. 判断是否可以表示为两个完全平方数之和
    max_a = int(math.isqrt(n))
    for a in range(max_a + 1):
        b_squared = n - a * a
        b = int(math.isqrt(b_squared))
        if b * b == b_squared:
            return 2  # 对应口诀3：两平方和验证返2
    
    # 4. 以上都不满足，则答案为 3
    return 3  # 对应口诀4：其余情况返回3
```

**解法2：**

```python
class Solution:
    def numSquares(self, n: int) -> int:
        # 初始化 dp 数组：dp[i] 表示组成 i 所需的最少完全平方数个数
        # 初始值设为无穷大（表示初始不可达），dp[0] = 0 是边界条件
        dp = [float('inf')] * (n + 1)
        dp[0] = 0
        
        # 遍历 1 到 n 的所有数字，计算每个数的最少完全平方数个数
        for i in range(1, n + 1):
            j = 1
            # 修正循环条件：j*j <= i 包含 j*j = i 的情况
            while j * j <= i:
                # 状态转移：dp[i-j²]+1 表示选 j² 这个数，加上组成 i-j² 的最少个数
                dp[i] = min(dp[i], dp[i - j * j] + 1)
                j += 1
        
        return dp[n]
```

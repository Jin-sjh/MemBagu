# 字符串解码

## 【问题】
请解释 字符串解码 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/decode-string/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772779292673-f8033d72-93f4-43ee-88f5-5a55f2f9483c.png)

## 【回答】
**口诀速记：**

利用栈保存嵌套编码结构中每一层的前缀字符串和重复次数，遇到闭合括号时弹出栈顶状态，将当前层字符串按次数重复后与前缀拼接，以此逐层处理嵌套的编码逻辑。

- 栈存串和重复数
- 初始化当前串与数
- 遇数字累积算数值
- 遇左括号存状态重置
- 遇右括号弹栈拼字符串
- 遇字母直接拼当前串
- 最终返回当前字符串

**代码实现：**

```python
def decodeString(s: str) -> str:
    stack = []  # 保存 (之前的字符串, 重复次数k)  # 对应口诀1：栈存串和重复数
    current_str = ""  # 当前正在构建的字符串
    current_num = 0   # 当前累积的数字k
    # 对应口诀2：初始化当前串与数
    
    for c in s:
        if c.isdigit():
            current_num = current_num * 10 + int(c)  # 处理多位数
            # 对应口诀3：遇数字累积算数值
        elif c == '[':
            # 保存当前状态，进入嵌套
            stack.append((current_str, current_num))  # 对应口诀4：遇左括号存状态重置（存状态）
            current_str = ""
            current_num = 0  # 对应口诀4：遇左括号存状态重置（重置）
        elif c == ']':
            # 弹出之前的状态，完成当前层的解码
            prev_str, num = stack.pop()
            current_str = prev_str + current_str * num  # 对应口诀5：遇右括号弹栈拼字符串
        else:  # 普通字母
            current_str += c  # 对应口诀6：遇字母直接拼当前串
    
    return current_str  # 对应口诀7：最终返回当前字符串
```

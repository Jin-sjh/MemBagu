# 正则表达式匹配

## 【问题】
请解释 正则表达式匹配 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/regular-expression-matching/

## 【回答】
**口诀速记：**

通过动态规划定义dp[i][j]表示字符串s前i个字符与模式p前j个字符的匹配状态，核心是分情况处理*（匹配零个前导字符或匹配一个 / 多个前导字符），并基于子问题的解推导当前状态，最终得到整个字符串与模式的匹配结果。

- 构建 (m+1)×(n+1) DP 表，dp [i][j] 表 s 前 i、p 前 j 字符是否匹配
- 初始化空串匹配：dp [0][0] 为真
- 处理 s 空 p 非空：p [j-1] 为 \* 时，dp [0][j] 继承 dp [0][j-2]（匹配零个）
- 遍历 s 和 p 的长度（从 1 开始）
- 非 \* 字符：s [i-1] 与 p [j-1] 相等 / 为.，则继承 dp [i-1][j-1]
- 遇 \* 字符：先继承 dp [i][j-2]（零匹配）
- 若 s [i-1] 与 p [j-2] 匹配，再或上 dp [i-1][j]（多匹配）
- 最终返回 dp [m][n] 判断整体匹配

**解法1：**

```python
def isMatch(s: str, p: str) -> bool:
    m, n = len(s), len(p)
    # 创建DP表，dp[i][j]表示s前i个字符和p前j个字符是否匹配
    dp = [[False] * (n + 1) for _ in range(m + 1)]  # 对应口诀1：构建(m+1)×(n+1) DP表
    dp[0][0] = True  # 空串和空模式匹配  # 对应口诀2：初始化空串匹配
    
    # 初始化：处理s为空，p不为空的情况
    for j in range(2, n + 1):
        if p[j-1] == '*':
            dp[0][j] = dp[0][j-2]  # 对应口诀3：处理s空p非空
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):  # 对应口诀4：遍历s和p的长度
            if p[j-1] != '*':
                # 情况1：当前字符不是'*'，直接匹配
                if s[i-1] == p[j-1] or p[j-1] == '.':
                    dp[i][j] = dp[i-1][j-1]  # 对应口诀5：非*字符匹配逻辑
            else:
                # 情况2：当前字符是'*'
                # 子情况2.1：'*'匹配零个前面的字符，忽略p[j-2]和p[j-1]
                dp[i][j] = dp[i][j-2]  # 对应口诀6：*匹配零个
                # 子情况2.2：'*'匹配至少一个前面的字符，检查s[i-1]是否与p[j-2]匹配
                if s[i-1] == p[j-2] or p[j-2] == '.':
                    dp[i][j] |= dp[i-1][j]  # 对应口诀7：*匹配多个

    return dp[m][n]  # 对应口诀8：返回整体匹配结果
```

**解法2：**

```python
def isMatch(s: str, p: str) -> bool:
    m, n = len(s), len(p)
    # 用一维数组优化空间
    dp = [False] * (n + 1)
    dp[0] = True

    # 初始化：处理s为空，p不为空的情况
    for j in range(2, n + 1):
        if p[j-1] == '*':
            dp[j] = dp[j-2]

    for i in range(1, m + 1):
        # 注意：需要从后往前更新，避免覆盖需要的值
        new_dp = [False] * (n + 1)
        for j in range(1, n + 1):
            if p[j-1] != '*':
                if s[i-1] == p[j-1] or p[j-1] == '.':
                    new_dp[j] = dp[j-1]
            else:
                new_dp[j] = new_dp[j-2]  # 匹配零个
                if s[i-1] == p[j-2] or p[j-2] == '.':
                    new_dp[j] |= dp[j]  # 匹配至少一个
        dp = new_dp

    return dp[n]
```

# 相交链表

## 【问题】
请解释 相交链表 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/intersection-of-two-linked-lists/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772775090126-7d61eb2b-6635-4e72-ad55-5725aed57495.png)

## 【回答】
**口诀速记：**

利用双指针交替遍历两个链表，让两指针走过的总路程相等，从而在相交点（或链表末尾）相遇，以此找到链表的相交起始节点。【 双针互走对方路，相遇即是相交处  】

**代码实现：**

```python
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None

class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> ListNode:
        if not headA or not headB:
            return None
        
        pA, pB = headA, headB
        
        while pA != pB:
            # pA 遍历完 A 后，从 B 开始
            pA = pA.next if pA else headB
            # pB 遍历完 B 后，从 A 开始
            pB = pB.next if pB else headA
        
        return pA
```

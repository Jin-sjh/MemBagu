# 接雨水

## 【问题】
请解释 接雨水 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/trapping-rain-water/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772724273760-9660f29d-1d3c-4afe-910d-3032fdca3a67.png)

## 【回答】
**口诀速记：**

通过确定每个位置左右两侧的最大高度，计算出该位置能承接的雨水量，再将所有位置的雨水量累加得到总雨水量。

- 空数组直接返回 0，初始化左右指针（左 0、右数组末尾）
- 初始化左右最大高度、结果为 0
- 左 < 右时循环：
- 左指针值 < 右指针值：✅ 左值≥左最大则更新左最大✅ 否则累加（左最大 - 左值）到结果，左指针右移
- 否则：✅ 右值≥右最大则更新右最大✅ 否则累加（右最大 - 右值）到结果，右指针左移
- 循环结束返回结果

**代码实现：**

```python
def trap(height):
    if not height:
        return 0  # 对应口诀1：空数组返回0
    
    left = 0
    right = len(height) - 1  # 对应口诀1：初始化左右指针
    left_max = right_max = 0
    res = 0  # 对应口诀2：初始化左右最大高度、结果
    
    while left < right:  # 对应口诀3：左<右时循环
        if height[left] < height[right]:
            if height[left] >= left_max:
                left_max = height[left]  # 对应口诀3：更新左最大
            else:
                res += left_max - height[left]  # 对应口诀3：累加储水量
            left += 1  # 对应口诀3：左指针右移
        else:
            if height[right] >= right_max:
                right_max = height[right]  # 对应口诀3：更新右最大
            else:
                res += right_max - height[right]  # 对应口诀3：累加储水量
            right -= 1  # 对应口诀3：右指针左移
    return res  # 对应口诀4：返回结果

# 测试示例
height = [0,1,0,2,1,0,1,3,2,1,2,1]
print(trap(height))  # 输出: 6
```

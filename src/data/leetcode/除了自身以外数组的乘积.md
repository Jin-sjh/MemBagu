# 除了自身以外数组的乘积

## 【问题】
请解释 除了自身以外数组的乘积 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/product-of-array-except-self/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772777094244-f14b5ea7-2dbc-4f00-882b-98d3f54041da.png)

## 【回答】
**口诀速记：**

通过两次线性遍历，分别计算每个元素的前缀乘积和后缀乘积，再将两者相乘，从而在不使用除法的情况下，得到每个元素除自身外的乘积。

- 初始化结果全为 1
- 前缀遍历存乘积
- 后缀逆序乘结果
- 最终返回结果数组

**代码实现：**

```python
def productExceptSelf(nums):
    n = len(nums)
    res = [1] * n  # 对应口诀1：初始化结果全为1
    
    # 计算前缀乘积
    prefix = 1
    for i in range(n):
        res[i] = prefix  # 对应口诀2：前缀遍历存乘积（存前缀）
        prefix *= nums[i]  # 对应口诀2：前缀遍历存乘积（更新前缀）
    
    # 计算后缀乘积，并与前缀乘积合并
    suffix = 1
    for i in range(n-1, -1, -1):
        res[i] *= suffix  # 对应口诀3：后缀逆序乘结果（乘后缀）
        suffix *= nums[i]  # 对应口诀3：后缀逆序乘结果（更新后缀）
    
    return res  # 对应口诀4：最终返回结果数组
```

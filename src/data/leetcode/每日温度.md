# 每日温度

## 【问题】
请解释 每日温度 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/daily-temperatures/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772782844375-0cb34138-d7c7-4216-ac04-e277cda5d432.png)

## 【回答】
**口诀速记：**

利用单调栈维护未找到下一个更高温度的日期索引，在一次遍历中以 O (n) 时间复杂度高效求解每个位置之后首个更大元素的距离。

- 初始化结果数组全 0
- 单调栈存索引保温度递减
- 遍历每个温度索引
- 栈非空且当前温更高时弹栈
- 计算弹出索引的等待天数
- 当前索引入栈
- 最终返回结果数组

**代码实现：**

```python
def dailyTemperatures(temperatures):
    n = len(temperatures)
    answer = [0] * n  # 对应口诀1：初始化结果数组全0
    stack = []  # 单调栈，存储索引，保证对应温度递减  # 对应口诀2：单调栈存索引保温度递减
    
    for i in range(n):  # 对应口诀3：遍历每个温度索引
        # 当前温度大于栈顶温度时，弹出并计算天数
        while stack and temperatures[i] > temperatures[stack[-1]]:
            j = stack.pop()  # 对应口诀4：栈非空且当前温更高时弹栈
            answer[j] = i - j  # 对应口诀5：计算弹出索引的等待天数
        stack.append(i)  # 对应口诀6：当前索引入栈
    
    return answer  # 对应口诀7：最终返回结果数组
```

# 搜索旋转排序数组

## 【问题】
请解释 搜索旋转排序数组 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/search-in-rotated-sorted-array/

## 【回答】
**口诀速记：**

利用二分查找，每次确定数组的左半段或右半段哪个是有序区间，然后判断目标值是否在该有序区间内，以此不断缩小查找范围，最终在 O (log n) 时间内找到目标值或确定其不存在。

- 初始化左右指针（左 0、右为数组末尾）
- 左≤右时循环，计算中间索引 mid
- nums [mid]==target 则返回 mid
- 判断左半区是否有序（nums [left]≤nums [mid]）
- 左半有序：target 在左区间则右 = mid-1，否则左 = mid+1
- 右半有序：target 在右区间则左 = mid+1，否则右 = mid-1
- 循环结束未找到，返回 - 1

**代码实现：**

```python
def search(nums, target):
    left, right = 0, len(nums) - 1  # 对应口诀1：初始化左右指针
    while left <= right:  # 对应口诀2：左≤右时循环
        mid = (left + right) // 2  # 对应口诀2：计算中间索引
        if nums[mid] == target:
            return mid  # 对应口诀3：找到目标返回mid
        # 判断左半部分是否有序
        if nums[left] <= nums[mid]:  # 对应口诀4：判断左半区有序
            # target 在左半有序区间内
            if nums[left] <= target < nums[mid]:
                right = mid - 1
            else:
                left = mid + 1  # 对应口诀5：左半有序的指针移动逻辑
        # 否则右半部分有序
        else:  # 对应口诀4：右半区有序
            # target 在右半有序区间内
            if nums[mid] < target <= nums[right]:
                left = mid + 1
            else:
                right = mid - 1  # 对应口诀6：右半有序的指针移动逻辑
    return -1  # 对应口诀7：未找到返回-1
```

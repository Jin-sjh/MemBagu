# 零钱兑换

## 【问题】
请解释 零钱兑换 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/coin-change/

## 【回答】
**口诀速记：**

用动态规划（完全背包模型），通过子问题的最优解（凑成较小金额的最少硬币数），逐步推导出凑成目标金额的最少硬币数。

- DP 数组初始无穷大
- 零金额步数设为 0
- 遍历金额逐枚查
- 币值小可更 DP 值
- 最终判值返结果

**代码实现：**

```python
def coinChange(coins, amount):
    # 初始化dp数组，长度为amount+1，初始值设为amount+1（表示无穷大）
    dp = [amount + 1] * (amount + 1)  # 对应口诀1：DP数组初始无穷大
    dp[0] = 0  #  base case  # 对应口诀2：零金额步数设为0
    
    for i in range(1, amount + 1):  # 对应口诀3：遍历金额逐枚查（遍历金额）
        for coin in coins:  # 对应口诀3：遍历金额逐枚查（逐枚查）
            if coin <= i:  # 对应口诀4：币值小可更DP值（币值小）
                dp[i] = min(dp[i], dp[i - coin] + 1)  # 对应口诀4：币值小可更DP值（更DP值）
    
    return dp[amount] if dp[amount] <= amount else -1  # 对应口诀5：最终判值返结果
```

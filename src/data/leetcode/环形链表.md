# 环形链表

## 【问题】
请解释 环形链表 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/linked-list-cycle/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772773485675-39c7b68b-2dbd-4b2d-8c9c-e87c0c57495e.png)

## 【回答】
**口诀速记：**

利用快慢指针（Floyd 判圈算法），通过让快指针每次走两步、慢指针每次走一步，若两指针最终相遇则证明链表存在环，否则无环。

- 初始化快慢指针：慢指针slow、快指针fast均指向链表头节点head
- 循环遍历链表（条件：快指针fast和fast.next均不为空，避免空指针异常）：
- 慢指针向前走 1 步（slow = slow.next）
- 快指针向前走 2 步（fast = fast.next.next）
- 若快慢指针指向同一个节点（相遇），说明链表有环，直接返回 True
- 若循环结束（快指针走到链表末尾），说明链表无环，返回 False

**代码实现：**

```python
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None

class Solution:
    def hasCycle(self, head: ListNode) -> bool:
        # 初始化快慢指针
        slow = head  # 对应口诀1：慢指针初始指向头节点
        fast = head  # 对应口诀1：快指针初始指向头节点
        
        while fast and fast.next:  # 对应口诀2：循环条件（快指针未到末尾）
            slow = slow.next       # 慢指针走一步  # 对应口诀2：慢指针走1步
            fast = fast.next.next  # 快指针走两步  # 对应口诀2：快指针走2步
            
            if slow == fast:       # 相遇则有环  # 对应口诀2：指针相遇，返回True
                return True
        
        # 快指针走到末尾，无环
        return False  # 对应口诀3：无环返回False
```

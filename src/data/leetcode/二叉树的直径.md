# 二叉树的直径

## 【问题】
请解释 二叉树的直径 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/diameter-of-binary-tree/

![题目图示](https://cdn.nlark.com/yuque/0/2026/png/27223229/1772781559846-950af47a-7f4e-4214-a2a0-37884f1bdd29.png)

## 【回答】
**口诀速记：**

采用后序遍历自底向上计算每个节点的左右子树深度，同时将 “左右深度之和” 的最大值作为二叉树的直径。

- 初始化最大直径为 0
- 递归求节点深度
- 空节点返回深度 0
- 求左右子树深度
- 更新最大直径为左右深度和
- 返回当前节点最大深度 + 1
- 最终返回最大直径

**代码实现：**

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def diameterOfBinaryTree(self, root: TreeNode) -> int:
        self.max_diameter = 0  # 对应口诀1：初始化最大直径为0
        
        def depth(node):
            if not node:
                return 0  # 对应口诀3：空节点返回深度0
            left_depth = depth(node.left)
            right_depth = depth(node.right)  # 对应口诀4：求左右子树深度
            # 更新最大直径
            self.max_diameter = max(self.max_diameter, left_depth + right_depth)  # 对应口诀5：更新最大直径为左右深度和
            # 返回当前节点的深度
            return max(left_depth, right_depth) + 1  # 对应口诀6：返回当前节点最大深度+1
        
        depth(root)  # 对应口诀2：递归求节点深度
        return self.max_diameter  # 对应口诀7：最终返回最大直径
```

# 多数元素

## 【问题】
请解释 多数元素 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/majority-element/

## 【回答】
**口诀速记：**

利用摩尔投票法的 “抵消” 思想，通过一次遍历，在 O (1) 空间内找到出现次数超过数组长度一半的元素。

【 归零换候选，相同加一不同减  】

**代码实现：**

```python
def majorityElement(nums):
    count = 0
    candidate = None
    
    for num in nums:
        if count == 0:
            candidate = num
        count += (1 if num == candidate else -1)
    
    return candidate
```

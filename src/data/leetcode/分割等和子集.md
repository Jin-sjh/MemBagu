# 分割等和子集

## 【问题】
请解释 分割等和子集 的解题思路和实现方法。

题目链接: https://leetcode.cn/problems/partition-equal-subset-sum/

## 【回答】
**口诀速记：**

将 “分割等和子集” 问题转化为 “是否存在和为总和一半的子集”，再用 0-1 背包的动态规划思路，通过一维数组从后往前更新，判断能否凑出目标和。

- 总和奇数直接返假
- 目标值为总和半值
- DP 数组初始假，零位设真
- 逆序遍历更 DP 值
- 最终返目标位结果

**代码实现：**

```python
def canPartition(nums):
    total = sum(nums)
    if total % 2 != 0:
        return False  # 对应口诀1：总和奇数直接返假
    target = total // 2  # 对应口诀2：目标值为总和半值
    dp = [False] * (target + 1)
    dp[0] = True  # 对应口诀3：DP数组初始假，零位设真
    
    for num in nums:
        # 从后往前遍历，避免重复使用同一个元素
        for i in range(target, num - 1, -1):  # 对应口诀4：逆序遍历更DP值（逆序遍历）
            if dp[i - num]:
                dp[i] = True  # 对应口诀4：逆序遍历更DP值（更DP值）
    return dp[target]  # 对应口诀5：最终返目标位结果
```

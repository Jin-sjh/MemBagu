# 数组方法

## 【问题】
分类列举 JavaScript 数组方法，并说明哪些会改变原数组

## 【回答】

### 会改变原数组的方法

- `push()` - 末尾添加元素
- `pop()` - 删除末尾元素
- `shift()` - 删除首元素
- `unshift()` - 开头添加元素
- `splice()` - 添加/删除元素
- `sort()` - 排序
- `reverse()` - 反转数组

### 不改变原数组的方法

- `concat()` - 合并数组
- `slice()` - 截取数组
- `join()` - 连接为字符串
- `map()` - 映射新数组
- `filter()` - 过滤数组
- `reduce()` - 累计计算
- `some()` - 测试某些元素
- `every()` - 测试所有元素
- `find()` - 查找元素
- `findIndex()` - 查找索引
- `includes()` - 是否包含

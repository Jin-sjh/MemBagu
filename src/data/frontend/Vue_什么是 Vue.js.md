# Vue 什么是 Vue.js

## 【问题】
什么是 Vue.js？

## 【回答】
Vue (发音为 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式 JavaScript 框架。

**构建用户界面**：Vue 的核心库只关注视图层（宁在网页上看到的部分）。它使用"模板"将数据渲染到 DOM (网页)。

**渐进式框架**：这是 Vue 的核心理念。

a. **易于上手**：宁可以像使用 jQuery 一样，仅通过 `<script>` 标签将其引入现有项目，用于增强某个页面。

b. **逐步扩展**：当应用变得复杂时，宁可以引入 Vue Router（路由）、Pinia（状态管理）等官方工具，构建一个功能完备的单页应用 (SPA)。

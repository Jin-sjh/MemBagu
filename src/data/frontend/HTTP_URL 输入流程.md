# 输入 URL 会发生什么

## 8. 输入 URL 会发生什么

### 【问题】
输入 URL 会发生什么

### 【回答】

**1. DNS 解析：** 浏览器会先解析 URL 中的域名，获得服务器的 IP 地址。

**2. TCP 连接：** 通过三次握手（如果是 HTTPS，还会进行 SSL/TLS 握手）建立 TCP 连接。

**3. 发送请求：** 浏览器根据请求方法（GET、POST 等）将数据发送到服务器。

**4. 服务器响应：** 服务器处理请求，返回状态码和响应数据。

**5. 渲染页面：** 浏览器根据响应的 HTML、CSS、JS 等文件进行页面渲染。

**6. 关闭连接：** TCP 连接通过四次挥手被关闭。

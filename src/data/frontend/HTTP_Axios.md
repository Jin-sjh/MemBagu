# Axios 详解

## 1. Axios 是什么？解决了什么问题？

### 【问题】
Axios 是什么？解决了什么问题？

### 【回答】
Axios 是一个基于 Promise 的 HTTP 客户端，支持浏览器和 Node.js，对原生 XHR / http 模块进行了封装，提供了统一的请求 API、拦截器机制、自动转换数据、请求取消、并发控制等能力，适合在工程化前端项目中使用。

### 【隐含加分点】
- 浏览器端：基于 XMLHttpRequest
- Node 端：基于 http / https 模块
- 核心目标：统一请求层、可扩展、可维护

## 2. axios 请求失败处理方法

### 【问题】
axios 请求失败处理方法

### 【回答】
我在项目中会封装 axios 统一处理请求异常：

首先创建实例配置超时和基础路径，通过响应拦截器捕获网络错误、超时、4xx/5xx 及业务错误；

然后分类处理：网络异常提示网络问题，401 跳登录，403 提示无权限，服务端错误统一提示；

同时会做自动重试、取消重复请求、错误日志上报，最后返回标准化结果，保证代码健壮和用户体验。

# TCP 队头阻塞

## 【问题】
为什么会出现 TCP 的队头阻塞

## 【回答】
队头阻塞是指当一个 TCP 连接中的一个数据包丢失或延迟，后续的数据包必须等到这个数据包被正确接收后才能继续处理。即使其他数据包已经到达，仍需要等待队头的数据包。

### 原因：
- **可靠性**：TCP 为了确保数据的可靠传输，会等待丢失的数据包被重新传输并按顺序接收，导致后续数据的延迟。

### 解决方案：
- **HTTP/2** 通过多路复用和流优先级来解决 TCP 队头阻塞，但仍然受到 TCP 固有问题的影响。
- **HTTP/3** 通过 QUIC 协议利用 UDP 减少了队头阻塞的影响。

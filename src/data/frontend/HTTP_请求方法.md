# HTTP 请求方法

## 6. 常见的 HTTP 请求方法

【问题】GET 方法的作用是什么？

【回答】用于请求获取资源。

【问题】POST 方法的作用是什么？

【回答】用于提交数据，通常用于创建或修改资源。

【问题】PUT 方法的作用是什么？

【回答】用于更新资源。

【问题】DELETE 方法的作用是什么？

【回答】用于删除资源。

【问题】HEAD 方法的作用是什么？

【回答】类似于 GET，但只请求头部。

【问题】OPTIONS 方法的作用是什么？

【回答】用于获取支持的请求方法或查询服务器支持的选项。

【问题】PATCH 方法的作用是什么？

【回答】用于部分更新资源。

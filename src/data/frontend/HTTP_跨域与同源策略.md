# 9. 跨域与同源策略

## 【问题】什么是同源策略？

## 【回答】
浏览器为保证安全性，规定不同域之间的网页不能互相访问。只有在协议、域名、端口都相同的情况下，才能进行交互。

## 【问题】什么是跨域问题？常见的跨域解决方法有哪些？

## 【回答】
当一个网页试图访问另一个域的资源时，就会遇到跨域问题。常见的跨域解决方法包括：

1. **CORS（跨源资源共享）**：通过设置服务器的 `Access-Control-Allow-Origin` 头部，允许特定域的访问。

2. **JSONP**：通过 `<script>` 标签绕过浏览器的同源策略，通常用于 GET 请求。

3. **WebSocket**：通过 WebSocket 协议进行跨域通信。

4. **代理**：通过设置代理服务器来绕过同源策略。

# CSS 层叠与优先级

---

## 问题 1：CSS 的三种引入方式？

### 【问题】
CSS 的三种引入方式？

### 【回答】
1. 行内样式：写在 HTML 元素的 `style` 属性里。

2. 内部样式表：在 HTML 文档的 `<style>` 标签中定义。

3. 外部样式表：单独写在 `.css` 文件中，通过 `<link>` 引入。

## 问题 2：浏览器如何决定某个元素最终样式？

### 【问题】
浏览器如何决定某个元素最终样式？

### 【回答】
浏览器依据 **层叠（Cascade）** 机制计算最终样式：先收集所有样式，再按 **权重（Specificity）** 判定优先级（ID > 类 / 伪类 > 元素 / 伪元素）；权重相同时，按 **源码顺序（Source Order）**，后定义的覆盖先定义的；行内样式权重高于普通规则，`!important` 优先级最高。

## 问题 3：什么是样式冲突？谁赢？

### 【问题】
什么是样式冲突？谁赢？

### 【回答】
样式冲突是多个 CSS 规则同时作用于同一元素的同一属性时产生的样式冲突。解决规则为：**权重（Specificity）高**的规则胜出，权重顺序为 `!important` > 行内样式 > ID 选择器 > 类 / 伪类 / 属性选择器 > 元素 / 伪元素选择器；**权重相同时，后定义的（源码顺序）样式赢**，核心是 CSS 层叠机制。

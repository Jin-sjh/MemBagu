# Soft Prompt 和 Hard Prompt

## 【问题】

什么是 Soft Prompt 和 Hard Prompt？它们有什么区别？

## 【回答】

### 定义

在大型语言模型 (LLM) 的世界里，**Prompt** 是指你给模型提供的输入文本，目的是引导模型生成特定的输出。根据 Prompt 的形式和优化方式，它们可以分为两大类：**Hard Prompt（硬提示）** 和 **Soft Prompt（软提示）**。

### Hard Prompt（硬提示）

**Hard Prompt** 就是指那些**人类可读的、明确的、离散的文本指令**。它们通常是自然语言句子、短语或关键词，直接告诉模型要执行什么任务或生成什么内容。我们一般接触的 Prompt 都是 Hard Prompt。

**特点**：
- 人类可读
- 明确的文本指令
- 离散的自然语言
- 直接编写

### Soft Prompt（软提示）

**Soft Prompt** 就是指**不可读的、可学习的连续向量表示**，它们在模型的嵌入空间中进行优化。与 Hard Prompt 不同，Soft Prompt 不是由人类直接编写的文本，而是通过算法训练出来的、附加到模型输入中的数值向量。

**Soft Prompt** 像是给这个人悄悄地做了一些"脑力训练"，让他自然而然地在执行特定任务时，就具备了你希望的思维模式和偏好，而你不需要每次都给他书面说明。

**特点**：
- 不可读的向量表示
- 可学习的连续向量
- 在嵌入空间中优化
- 通过算法训练得到

### 核心区别

| 特性 | Hard Prompt | Soft Prompt |
|------|-------------|-------------|
| 可读性 | 人类可读 | 不可读 |
| 形式 | 离散文本 | 连续向量 |
| 编写方式 | 人工编写 | 算法训练 |
| 优化空间 | 自然语言空间 | 嵌入向量空间 |
| 灵活性 | 每次需明确说明 | 一次训练持续生效 |

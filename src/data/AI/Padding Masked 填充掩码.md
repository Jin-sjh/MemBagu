# Padding Masked（填充掩码）是什么？

**Padding Masked** 是深度学习（尤其是 Transformer、大语言模型）中处理**变长序列**的关键技术，全称是 **Padding Mask（填充掩码）**，"Masked"表示"被掩码的"状态。

## 核心作用

在自然语言处理（NLP）、语音识别等任务中，输入的句子 / 序列长度往往不一致：

- 为了让模型能**批量处理**数据，需要把短序列用无意义的填充符（如 `<pad>`、`0`）补齐到统一长度。
- 这些填充符本身没有语义信息，如果让模型关注它们，会引入噪声、干扰学习。
- **Padding Mask** 的作用就是：**标记这些填充位置，让模型在计算注意力时完全忽略它们**。

## 工作原理（简化版）

1. **标记无效位置**：生成一个掩码矩阵，真实内容位置标记为 `False`（或 `0`），填充位置标记为 `True`（或 `1`）。

2. **屏蔽注意力分数**：在计算注意力权重（`Q*K^T`）后，将填充位置对应的分数设置为**极小的负数**（如 `-1e9`）。

3. **Softmax 归一化**：经过 Softmax 后，填充位置的权重会被压缩到接近 `0`，模型就不会关注这些无效位置了。

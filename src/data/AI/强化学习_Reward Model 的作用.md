# Reward Model 的作用

## 【问题】
Reward Model 用于计算生成 token 的即时收益，它就是 RW 阶段所训练的奖励模型，在 RLHF 过程中，它的参数是冻结的。有一个问题，既然已经拿 Critic Model 预测出了未来的总收益 V（包括了即时和未来），为什么还需要 Reward Model 呢？直接用 V_t 不就好了吗？

## 【回答】
但事实上是不行的，可以看下价值函数的定义：

$$V_t = R_t + \gamma V_{t+1}$$

这个函数告诉我们，我们当前可以用两个结果来表示 t 时刻的总收益：

- 结果 1：Critic 模型预测的 V_t。
- 结果 2：Reward 模型预测的 R_t 和 critic 模型预测的 V_{t + 1}。

那么哪一个结果更靠近上帝视角给出的客观值呢？当然是结果 2，因为结果 1 全靠预测，而结果 2 中的 R 是事实数据。

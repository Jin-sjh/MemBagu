# Dense Model 稠密模型

## 【问题】

什么是 Dense Model 稠密模型？它的核心特点是什么？

## 【回答】

**Dense Model** 是目前大多数主流大模型（如早期的 BERT、GPT 系列、Llama 系列等）采用的经典架构。

### 核心特点

1. **所有参数都被激活**
   - 在 Dense Model 中，模型的所有参数（包括所有的层、所有的神经元）在处理每个输入（例如一个词或一个序列）时都会被激活并参与计算。

2. **计算量与参数量成正比**
   - 当模型规模（参数量）增大时，每次前向传播的计算量也会相应地线性增加。
   - 这意味着更大的模型需要更多的计算资源（GPU/TPU、内存）和更长的推理时间。

3. **模型复杂度高**
   - 随着模型参数的增加，Dense Model 的训练和部署成本会变得非常高昂，有时甚至难以承受。

4. **优点**
   - 在给定参数量和计算预算下，Dense Model 通常能够学习到非常密集和丰富的特征表示，从而在各种任务上表现出强大的泛化能力和性能。

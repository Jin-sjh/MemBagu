# 5.1 为什么需要"先升维再降维"？

## 【问题】
为什么需要"先升维再降维"？

## 【回答】

### 1. 增强表达能力
低维空间（如 768 维）中，许多复杂特征是线性不可分的。**升维后，模型可以在高维空间中构造更复杂的决策边界**（类似 SVM 中的核技巧）。

### 2. 特征筛选与重构

**升维阶段**：展开特征，暴露隐藏的关联（例如"苹果"和"iPhone"的潜在联系）。

**降维阶段**：通过权重矩阵 W2 的稀疏性，自动保留重要特征（如"苹果"作为水果的语义）。

### 3. 与注意力机制互补
注意力层擅长捕捉 token 间的关系，但输出可能是冗余或噪声较多的混合特征。

FFN 通过非线性变换，对每个 token 的表示进行独立精炼，类似"去噪 + 提纯"。

# 八股转音频功能开发计划

## 项目概述

为现有的八股记忆系统添加「一键转音频」功能，将 Markdown 格式的八股内容转换为 MP3 音频文件，支持单独文件和合集两种模式，方便用户通过倾听方式记忆八股内容。

## 功能需求

### 核心功能

1. **单独文件模式**：每个问题生成独立的 MP3 文件，按分类组织
2. **合集模式**：将所有内容合并为一个完整的 MP3 文件
3. **Vue 界面集成**：提供可视化操作界面

### 用户场景

* 通勤/运动时通过音频复习八股

* 一键导出完整合集分享

* 按分类选择性学习

## 技术方案

### TTS 服务选型

| 方案             | 优点                  | 缺点        | 推荐    |
| -------------- | ------------------- | --------- | ----- |
| **edge-tts**   | 免费、高质量中文语音、直接输出 MP3 | 依赖网络      | ⭐⭐⭐⭐⭐ |
| 百度语音合成         | 稳定                  | 需注册、有配额限制 | ⭐⭐⭐   |
| Web Speech API | 浏览器内置               | 无法生成 MP3  | ⭐⭐    |

**选择方案**：edge-tts（基于 Microsoft Edge 的免费 TTS 服务）

## API 接口设计

### 生成单独音频

```
POST /api/audio/generate
Request: { categories, options }
Response: SSE 流
  data: {"progress": 10, "file": "xxx.md"}
  data: {"progress": 20, "file": "yyy.md"}
  ...
  data: {"done": true, "files": [...]}
```

### 生成合集音频

```
POST /api/audio/collection
Request: { categories, options }
Response: SSE 流
  data: {"progress": 50, "status": "generating"}
  data: {"progress": 80, "status": "merging"}
  data: {"done": true, "url": "/api/audio/download/xxx"}
```

### 下载音频

```
GET /api/audio/download/:filename
Response: audio/mpeg 文件流
```

## 配置选项

### 可用中文语音

| 语音 ID                | 名称 | 风格          |
| -------------------- | -- | ----------- |
| zh-CN-XiaoxiaoNeural | 晓晓 | 女声，自然流畅（推荐） |
| zh-CN-YunxiNeural    | 云希 | 男声，亲切       |
| zh-CN-YunyangNeural  | 云扬 | 男声，新闻播报风格   |
| zh-CN-XiaoyiNeural   | 晓伊 | 女声，活泼       |

### 语速设置

* `-50%` 到 `+100%`

* 默认：`+0%`（正常语速）

## 注意事项

1. **网络依赖**：edge-tts 需要网络连接
2. **生成时间**：大量文件转换需要较长时间，需显示进度
3. **并发控制**：控制并发请求数，避免被限流
4. **错误处理**：网络错误时自动重试
5. **文件管理**：定期清理临时文件

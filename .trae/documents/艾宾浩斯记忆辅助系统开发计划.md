# 艾宾浩斯记忆辅助系统开发计划

## 项目概述

基于 Vue 3 开发一个八股文背诵辅助系统，利用艾宾浩斯遗忘曲线原理，帮助用户科学复习和记忆八股知识点。

## 当前项目分析

### 文件结构
- 13个八股文件，涵盖 CSS 和 JavaScript 主题
- 文件命名格式：`分类_主题.md`

### 文件格式规范
```markdown
# 标题

## 【问题】
问题内容

## 【回答】
回答内容

## 【问题】
下一个问题...

## 【回答】
下一个回答...
```

## 艾宾浩斯记忆曲线

复习间隔时间点：
| 复习次数 | 间隔时间 |
|---------|---------|
| 第1次 | 20分钟 |
| 第2次 | 1小时 |
| 第3次 | 9小时 |
| 第4次 | 1天 |
| 第5次 | 2天 |
| 第6次 | 6天 |
| 第7次 | 30天 |

## 技术栈

- **前端框架**: Vue 3 + Composition API
- **构建工具**: Vite
- **样式**: 原生 CSS / CSS Modules
- **数据存储**: LocalStorage（持久化学习进度）
- **Markdown解析**: 自定义解析器（解析【问题】【回答】格式）

## 核心功能模块

### 1. 数据解析模块
- 扫描并解析所有 `.md` 文件
- 提取【问题】和【回答】内容
- 生成结构化数据（题目ID、分类、问题、回答）

### 2. 学习进度管理模块
- 记录每道题的学习状态
- 计算下次复习时间
- 存储复习历史记录

### 3. 复习提醒模块
- 显示今日待复习题目
- 按紧急程度排序
- 可视化复习进度

### 4. 提问练习模块
- 显示问题，隐藏答案
- 用户自我评估（记住/未记住）
- 根据评估结果调整复习计划

### 5. 统计分析模块
- 学习进度统计
- 记忆曲线可视化
- 各分类掌握程度

## 用户交互流程

1. **首次使用**：系统自动解析所有八股文件，初始化学习数据
2. **开始学习**：选择分类，系统展示未学习或待复习的题目
3. **答题过程**：
   - 显示问题，用户思考
   - 点击显示答案
   - 用户自评：记住/未记住
   - 系统记录并计算下次复习时间
4. **复习提醒**：首页显示今日待复习数量，按紧急程度排序
5. **进度查看**：统计页面展示学习进度和记忆曲线

## 注意事项

1. 所有数据存储在 LocalStorage，刷新不丢失
2. 支持导入/导出学习进度（可选功能）
3. 响应式设计，支持移动端使用
4. 简洁美观的 UI，专注学习体验
